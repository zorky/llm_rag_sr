Official OMIs Reference

This topic presents the list of supported and deprecated official OUTSCALE Machine Images (OMIs) for each Region.

Official OMIs use Cloud-Init. For more information, see the official Cloud-Init documentation.

3DS OUTSCALE ensures that the content of official OMIs is up-to date. However, once a VM is created using one of these OMIs, you are responsible for updating your run-time environment.

OOS Tools Overview

To use OUTSCALE Object Storage (OOS), different tools compatible with the Simple Storage Service (S3) API are available.

AWS CLI

AWS CLI, a command-line interface developed by Amazon Web Services, enables you to manage your OOS resources. In particular, you can use the s3api commands and s3 commands of AWS CLI. For more information about how to install AWS CLI, see Installing and Configuring AWS CLI.

This documentation primarily uses s3api commands. Corresponding s3 commands are also provided in tip boxes when available. For more information, see OOS API Reference.

S3cmd

s3cmd, an open-source command-line tool developed in Python, enables you to manage your OOS resources. It supports a variety of S3-compatible operations. For more information, see s3cmd.

GUIs

Graphical user interfaces (GUIs) exist to manage OOS resources. For example:

  • Cyberduck:

    • Available for macOS and Windows.

    • An open source client that enables you to directly browse your OOS resources on the OUTSCALE Cloud. It also enables you to edit your objects in a seamless way using any editor on your computer. For more information, see Configuring Cyberduck for OOS.

  • ExpanDrive:

    • Available for Linux, macOS, and Windows.

    • A commercial application that enables you to mount buckets as network drives on your computer. For more information, see the official ExpanDrive website.

  • TntDrive:

    • Available for Windows only.

    • A commercial application that enables you to mount buckets as network drives on your computer. For more information, see the official TntDrive website.

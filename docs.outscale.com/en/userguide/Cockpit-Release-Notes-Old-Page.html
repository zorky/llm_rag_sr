Cockpit v2 Release Notes (Old Page)

This page lists the new features contained in each release of Cockpit v2.

2024

June 17, 2024

As of June 17, 2024, the release notes are located on a new page: Release Notes.

April 23, 2024 (Cockpit 2.16)

  • The following features are now available in Cockpit v2:

    • EIM Groups:

      • Adding or Removing a User from a Group

      • Getting Information About Your Groups

    • EIM Policies:

      • Listing Managed Policies Attached to a Group

March 14, 2024 (Cockpit 2.15)

  • The following features are now available in Cockpit v2:

    • API Logs:

      • Getting Information About Your API Logs

    • Consumption:

      • Getting the Price of Your Resources

  • TINA v3 CPU generation are no longer available when creating a VM. For more information, see End-of-Life Policy.

  • You can now add tags to DHCP options. For more information, see About Tags.

  • A new graphic charter is available on Cockpit v2.

February 20, 2024 (Cockpit 2.14)

  • The following features are now available in Cockpit v2:

    • API Access Rules:

      • Creating an API Access Rule

      • Getting Information About Your API Access Rules

      • Modifying an API Access Rule

      • Deleting an API Access Rule

January 29, 2024 (Cockpit 2.13)

  • The following features are now available in Cockpit v2:

    • EIM Users:

      • Modifying an EIM Access Key

      • Deleting an EIM Access Key

    • EIM Policies:

      • Listing Managed Policies Attached to a User

2022

December 8, 2022 (Cockpit 2.3)

  • When creating a client gateway, you can now specify its name.

  • This release includes stability and error message improvements.

September 20, 2022 (Cockpit 2.2)

  • The Creating a VPN Connection feature is now available.

  • In Personal Space > My Consumption, you can now choose whether to show the overall consumption of your paying account and its linked accounts, or the individual consumption of your account. For more information, see Getting Information About Your Resource Consumption.

August 1, 2022 (Cockpit 2.1)

  • The following features are now available in Cockpit v2:

    • Creating a Route Table

    • Unlinking a Route Table from a Subnet

    • Deleting a Route Table

    • Deleting a VPN Connection

    • Creating a NAT Service

    • Deleting a NAT Service

June 20, 2022 (Cockpit 2.0)

  • Cockpit v2-beta is released. For more information about Cockpit v2, see About Cockpit > Cockpit v2-beta and for more information about available features, see Cockpit v2 Features Reference.

License Enrollment

Registering at First Boot

When booting a virtual machine (VM) for the first time with this Red Hat Enterprise Linux operating system, you must enroll your own licence. For more information about the procedure, see the Red Hat official website.

Or you can also use the following command:

$ subscription-manager register \
    --username <username> \
    --password <password> \
    --auto-attach

Using Cloud Init

You can use Cloud Init to configure your system during the installation and the first boot, and to enroll your own licence. For more information about the procedure, see the Red Hat official website.

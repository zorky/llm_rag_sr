The table below shows the correspondences between AWS-Compliant API methods and OUTSCALE API methods.

Two methods are considered equivalent if they can broadly serve the same purpose. However, their parameters do not necessarily have the exact same scopes or behaviors.

For more information about all the APIs, see About the APIs.

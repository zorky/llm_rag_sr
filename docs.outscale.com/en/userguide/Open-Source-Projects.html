Open Source Projects

3DS OUTSCALE maintains its open source projects on the GitHub platform.

The main GitHub organization for the projects is https://github.com/outscale.

Maturity Levels

Each project contains a badge in its README that identifies its maturity level among the following ones:

  • Graduated projects are considered stable, widely adopted, and production-ready. Developers are actively watching issues.

  • Incubating projects are used successfully in production by a small number of users. Developers can help resolve issues.

  • Sandbox projects are experiments not yet widely tested in production, and on the bleeding edge of technology.

  • Archived projects have reached the end of their lifecycle and have become inactive.

Estimating the Cost of a Product

As an OUTSCALE Marketplace visitor or buyer, you can estimate the cost of an OUTSCALE machine image (OMI) product on one or more virtual machines (VMs) before subscribing to it.

You can choose to estimate the cost of the product on OUTSCALE virtual machines (VMs) recommended by the seller partner, or custom OUTSCALE VMs. The estimation can cover the cloudgouv-eu-west-1 Region, and possibly eu-west-2 if the product has been published in this second Region.

  • This estimation does not constitute a quote or an invoice. It is valid for a limited time and the rate is subject to change.

  • To obtain a valid estimation for the Nuage public (UGAP) market, see Subscribing To a Nuage public Offer (UGAP).

  • The costs indicated for the VM reservation do not include storage. For more information, see Block Storage Unit (BSU) and 3DS OUTSCALE’s official price list.

  • When estimating the cost of a product based on a commitment period, you estimate the cost of the product used on one or more reserved VMs. For more information about reserved VMs and associated discounts, see About VMs > Reserved VMs and our official website.

  • For more information about product pricing, see OMI Products Pricing.

To estimate the cost of a product:

  1. In the Marketplace interface, click the Catalog tab.
    The product catalog appears.

  2. Use the filters on the left-hand side to find the desired product.

  3. Click the desired product card.
    The product page appears.

  4. If you want to subscribe to an offer on a recurrent basis:

    1. In the Software section, select the desired offer if there are several.

      • The commitment duration and billing frequency of the reservation and of the subscription offer are automatically aligned based on the selected offer.

      • You can directly proceed to the step 7 of the procedure.
    If you want to subscribe to an hourly and yearly offer per VM type:

    1. In the Software section, select the commitment duration, either On demand or 1 year.

    2. If you choose a one-year commitment, select the desired number of licenses.

      If you choose a one-year commitment period, the billing frequency is predefined. You can then directly proceed to the step 7 of the procedure.

    If you want to subscribe to another type of offer from the catalog, there is no required configuration at this step. You can directly proceed to the next step.

  5. In the VM Configuration section, select the commitment duration, either Without commitment, 1 month, 1 year, 2 years, or 3 years.

    The selected commitment duration determines the available billing frequencies available at the next step.

  6. Select the billing frequency, either At subscription, Monthly, or Yearly.

  7. Select the deployment Region, either cloudgouv-eu-west-1 or eu-west-2.

  8. Select a VM type, either Recommended VM or Custom VM.

    If you have selected Recommended VM:

    1. Select a VM recommended by the seller partner.

      The listed VM types are recommended by the seller partner for viable product deployment.

      For more information about a recommended VM, you can hover over the VM type in blue.
    If you have selected Custom VM:

    1. Configure the desired VM type.
      The VM type and prices are updated.

    For more information about VM types, see VM Types.

  9. Select the number of VMs.

    If you are subscribing to a yearly offer per VM type, the number of licenses that you have selected in step 4 determines the number of VMs. You do not have to select a number of VMs at this step.

    The software and VM costs are updated in the Estimation section on the right.

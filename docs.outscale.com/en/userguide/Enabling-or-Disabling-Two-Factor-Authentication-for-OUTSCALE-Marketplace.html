Enabling or Disabling Two-Factor Authentication for OUTSCALE Marketplace

When you have a Marketplace account, you can enable two-factor authentication (2FA) to increase the security of your account by requiring a one-time password (OTP) from an OTP application. You can disable 2FA for your account at any time.

For more information on Marketplace accounts, see About OUTSCALE Marketplace > Organization Creation and Login.

Enabling 2FA

Before you begin: Install an OTP application on a smartphone or on another device, preferably a different device than the one used to navigate the Marketplace.

Setting Up an Authentication Device

You can set up as many authentication devices as required.

  1. In the top right corner of the interface, click your username or email address.
    A drop-down menu appears.

  2. Click Authentication Parameters.
    An authentication management page is displayed.

  3. In the Two-factor authentication section, click Set up Authenticator application.
    The Mobile Authenticator Setup page is displayed.

  4. Open the OTP application on your device and scan the displayed barcode.

    If you are unable to scan the barcode, click Unable to scan? to display a key to manually type in your OTP application.

  5. In the One-time code field, type the one-time code provided by the application.

  6. In the Device Name field, type a name for the corresponding device.

  7. Click Submit.
    2FA is enabled for your account. You can log in to the Marketplace by typing an OTP generated by the corresponding device.

Disabling 2FA

Removing an Authentication Device

To completely disable 2FA for your account, you must delete all authentication devices.

  1. In the top right corner of the interface, click your username or email address.
    A drop-down menu appears.

  2. Click Authentication Parameters.
    An authentication management page is displayed.

  3. In the Two-factor authentication section, click Remove on the line of the device you want to remove.
    A confirmation dialog box is displayed.

  4. Click Continue.
    The authentication device is removed and you cannot log in using this device anymore.

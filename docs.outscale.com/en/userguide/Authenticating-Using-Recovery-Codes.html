Authenticating Using Recovery Codes

You can use a recovery code as a second factor authentication method to log in to Cockpit v2, in case your preferred multi-factor authentication (MFA) method is unavailable.

General Information

Recovery codes come in a list of 12. Each code can only be used once. A new list may be generated whether you have used all 12 codes or not. For more information, see Setting Up Recovery Codes.

Make sure to print, download, or copy the recovery codes carefully as you will not be able to retrieve them later.

Recovery codes may only be used instead of an MFA method.

Setting Up Recovery Codes

  1. Log in to Cockpit v2.

  2. In the top right corner, click the icon.
    A drop-down menu appears.

  3. Click  Authentication Parameters.
    The Authentication parameters page appears.

  4. In the Multi-factor Authentication section, click Set up recovery codes.
    A list of recovery codes appears.

    The list is only available during setup. Make sure to print, download, or copy the recovery codes carefully as you will not be able to retrieve them later.

  5. Click Set up.
    Recovery codes are set up for your account.

Using Recovery Codes

If you try to authenticate with an invalid recovery code four times in a row, further authentication attempts will be prevented for 1 minute. This locked time increases 1 minute every consecutive failed attempts, for up to 10 minutes.

  1. Access Cockpit v2.

  2. Select the Region where your OUTSCALE account is located.
    The Log in page appears.

  3. Authenticate using your credentials and click Log in.
    The Log in with a second factor page appears.

  4. Click Try another method.

  5. Click Recovery Code.

  6. In the Recovery code field, type the recovery code corresponding to the given number and click Log in.

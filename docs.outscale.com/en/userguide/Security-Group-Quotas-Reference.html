Security Group Quotas Reference

This page lists the quotas for security groups.

Quotas

By default, your account has the following quotas:

Quota Default value

Security groups (in the public Cloud)

100

Rules per security group (in the public Cloud)**

50

Security groups per virtual machine (VM)*

5

Security groups (in a Net)

100

Rules per security group (in a Net)**

50

If these default values are increased by the Support team:

  • The quota for Security groups per VM* multiplied by the quota for Rules per security group** cannot exceed 1000.

  • The quota for Security groups per VM* cannot exceed 16.

Examples

Security groups per VM Rules per security groups Total of rules for each VM

5 (by default)

50 (by default)

5 x 50 = 250 (by default)

10

100

10 x 100 = 1000 (maximum)

16 (maximum)

40 (lower than the default limit)

10 x 40 = 400

3 (lower than the default limit)

100

3 x 100 = 300

The Security group per VM quota counts the number of unique security groups on all network interfaces of a VM. This means that adding the same security group on several network interfaces of a VM will only count as 1.

For more information about your quotas, see Getting Information About Your Account and Quotas.

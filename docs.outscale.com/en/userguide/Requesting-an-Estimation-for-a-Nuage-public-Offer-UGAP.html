Requesting an Estimation for a Nuage public Offer (UGAP)

As a buyer within an organization belonging to the Nuage public market of the Union des groupements d’achats publics (UGAP), you must first request an estimation for an eligible Nuage public offer before you can subscribe to it.

You must then convert this estimation into a subscription in order to access the product. For more information, see Subscribing To a Nuage public Offer (UGAP).

You can request an estimation for:

  • Subscription offers dedicated to the Nuage public market, with or without virtual machine reservation.

  • Bring you own license (BYOl), free/open source, annual per VM type and Microsoft Windows licenses offers, necessarily with virtual machine reservation.

  • Mistral LLM services offers.

  • Your organization must be validated by 3DS OUTSCALE to be able to request an estimation for a subscription offer. For more information, see Creating an Organization as a Buyer.

  • Reserving one or more VMs commits you to paying for the resources consumed for the duration of the reservation and according to the selected billing frequency.

  • The costs indicated for the VM reservation do not include storage. For more information, see Block Storage Unit (BSU) and 3DS OUTSCALE’s official price list.

OMI Product Offer

To request an estimation for a Nuage public Subscription offer:

  1. In the Marketplace interface, click the Catalog tab.
    The product catalog appears.

  2. Use the filters on the left-hand side to find the desired product.

  3. Click the desired product card.
    The product page appears.

  4. Click the Request an estimation button.
    The Request an estimation page appears.

  5. If you want to subscribe to an offer on a recurrent basis:

    1. In the Software section, select the desired offer if there are several.

      • The commitment duration and billing frequency of the reservation and of the subscription offer are automatically aligned based on the selected offer.

      • You can directly proceed to the step 6. c. of the procedure.
    If you want to subscribe to a yearly offer per VM type:

    1. In the Software section, select the desired number of licenses.

      The commitment period and billing frequency are predefined for the Nuage public market. You can proceed to the step 6. c. of the procedure.

    If you want to subscribe to another type of offer from the catalog, there is no required configuration at this step. You can directly proceed to the next step.

  6. In the Virtual machine (VM) reservation section, specify whether you want to reserve one or more VMs.

    If you want to reserve one or more VMs:

    • Reserving one or more VMs commits you to paying for the resources consumed for the duration of the reservation and according to the selected billing frequency.

    • The costs indicated for the VM reservation do not include storage. For more information, see Block Storage Unit (BSU) and 3DS OUTSCALE’s official price list.

    1. Select the commitment duration, either 1 month, 1 year, 2 years, or 3 years.

      The selected commitment duration determines the available billing frequencies available at the next step.

    2. Select the billing frequency, either At subscription, Monthly, or Yearly.

    3. Select a Region and the OUTSCALE account to benefit from the VM reservation.

      • The listed Regions are those on which the product is available.

      • The listed OUTSCALE accounts are those that have been added to your organization. For more information, see Managing OUTSCALE Accounts as a Buyer.

    4. Select a VM recommended by the seller partner.

      Recommended VMs allow viable use of the product.

    5. Type the number of VMs to reserve, at least 1.

      If you are subscribing to a yearly offer per VM type, the number of licenses that you have selected in step 5 determines the number of VMs. This means you do not have to select a number of VMs at this stage.

  7. In the Order information section:

    1. Type the associated project’s name.

    2. Select the scheduled start date.

      The entered start date is not binding and may be modified at a later date.

  8. Click Request estimation.
    The estimation is sent to you by email and must be imported into the contract holder’s platform.

Mistral LLM Service Offer

A Mistral LLM service includes a large language model (LLM), a virtual machine (VM), and a flexible GPU (fGPU), deployed on an inference instance dedicated to a LLM.

To request an estimation for a Mistral LLM service offer:

  1. In the Marketplace interface, click the Catalog tab.
    The product catalog appears.

  2. Use the filters on the left-hand side to find the desired LLM service.

  3. Click the desired LLM service card.
    The LLM service page appears.

  4. Click Request an estimation.
    The Request an estimation page appears.

  5. In the Service configuration section, select the desired number of services.

    The deployment region, commitment duration and billing frequency are configured by default.

  6. In the Order information section:

    1. Type the associated project’s name.

    2. Select the scheduled start date.

      The entered start date is not binding and may be modified at a later date.

  7. Click Request estimation.
    The estimation is sent to you by email and must be imported into the contract holder’s platform.

For more information about deploying a Mistral LLM service once your subscription is validated, see Deploying a Mistral LLM Service.

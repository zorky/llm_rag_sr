Terraform

Terraform is a high-level tool developed by HashiCorp to orchestrate different Cloud providers infrastructure with a single configuration language.

3DS OUTSCALE is actively working on its own plugin based on the OUTSCALE API. For more information about its configuration, see the official Terraform provider documentation.

3DS OUTSCALE also provides AWS-compliant APIs with which you might be able to use another implemented provider, although support is limited.

General Information

Terraform allows you to create, update, and version your infrastructure in the OUTSCALE Cloud by describing it in configuration files.

You can access the OUTSCALE Terraform provider through the official website. For more information on the available resources, see the documentation.

This plugin does not cover all resources of the OUTSCALE API for the moment.

You can also directly download the plugin on the Github repository. You can open issues to report problems you encounter.

Before you begin, carefully read the README for provider declaration details.

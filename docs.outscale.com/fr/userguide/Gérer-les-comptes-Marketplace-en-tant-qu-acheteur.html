Gérer les comptes Marketplace en tant qu’acheteur

Le ou les administrateurs de votre organisation peuvent gérer des comptes Marketplace, c’est-à-dire les créer et les supprimer, et modifier leurs informations et/ou rôles. Pour en savoir plus sur les administrateurs, voir À propos de la Marketplace > Administrateurs et utilisateurs.

Créer un compte Marketplace

  1. Dans le coin supérieur droit de l’interface Marketplace, cliquez sur votre nom d’utilisateur ou sur votre adresse e-mail.
    Un menu déroulant apparaît.

  2. Cliquez sur Gestion des comptes Marketplace.
    La page Gestion des comptes Marketplace apparaît.

  3. Cliquez sur Créer un compte Marketplace.
    Un formulaire apparaît.

  4. Saisissez le prénom, le nom et l’e-mail associés au compte.

  5. Sélectionnez le rôle du compte, Utilisateur, Administrateur, ou les deux.

    Pour en savoir plus sur les rôles, voir À propos de la Marketplace > Administrateurs et utilisateurs.

  6. Cliquez sur Enregistrer.
    Le compte est créé et l’utilisateur correspondant reçoit un e-mail exigeant la création d’un mot de passe pour son compte.

Modifier un compte Marketplace

  1. Dans le coin supérieur droit de l’interface Marketplace, cliquez sur votre nom d’utilisateur ou sur votre adresse e-mail.
    Un menu déroulant apparaît.

  2. Cliquez sur Gestion des comptes Marketplace.
    La page Gestion des comptes Marketplace apparaît.

  3. Cliquez sur Modifier sur la ligne du compte que vous voulez modifier.
    Un formulaire apparaît.

  4. Modifiez les informations du compte.

  5. Cliquez sur Enregistrer.
    Le compte est modifié et l’utilisateur correspondant reçoit un e-mail l’informant de la modification.

Supprimer un compte Marketplace

  1. Dans le coin supérieur droit de l’interface Marketplace, cliquez sur votre nom d’utilisateur ou sur votre adresse e-mail.
    Un menu déroulant apparaît.

  2. Cliquez sur Gestion des comptes Marketplace.
    La page Gestion des comptes Marketplace apparaît.

  3. Cliquez sur Supprimer le compte Marketplace sur la ligne du compte que vous souhaitez supprimer.
    Une boîte de dialogue de confirmation apparaît.

  4. Cliquez sur Confirmer.
    Le compte est supprimé et l’utilisateur correspondant reçoit un e-mail l’informant de cette suppression.

    Les comptes supprimés ne peuvent plus se connecter à la Marketplace.

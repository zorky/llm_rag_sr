Démarrage rapide

Cette page de démarrage rapide vise à présenter le Cloud OUTSCALE ainsi que les outils pour l’utiliser.

Présentation

3DS OUTSCALE offre des solutions d’Infrastructure as a Service (IaaS), consistant à fournir des infrastructures informatiques à la demande. Les utilisateurs d’IaaS installent leurs propres applications et données dans les environnements virtuels fournis.

Les ressources fournies sont classées comme suit :

  • Ressources de calcul. Pour en savoir plus, voir Puissance de calcul.

  • Ressources de stockage. Pour en savoir plus, voir Stockage.

  • Ressources de réseau et sécurité. Pour en savoir plus, voir Réseau et sécurité.

Votre compte OUTSCALE détermine les quotas de ressources que vous pouvez consommer. Pour en savoir plus, voir Compte.

Outils pour gérer vos ressources

API

Les interfaces de programmation applicatives (API) sont le principal moyen de communiquer avec le Cloud OUTSCALE. Elles sont destinées à des utilisateurs ayant des connaissances en programmation. Les requêtes API peuvent être intégrées dans vos programmes et scripts, et sont particulièrement utiles à l’automatisation.

3DS OUTSCALE développe d’abord tous ses services pour les API. Celles-ci vous permettent donc d’exécuter toutes les actions publiquement disponibles.

Vous pouvez utiliser les API suivantes :

  • L’API OUTSCALE développée par 3DS OUTSCALE.

  • Les API AWS-compliant développées par 3DS OUTSCALE, fournissant une interopérabilité avec les API d’Amazon Web Services.

Pour en savoir plus, voir À propos des API.

SDK et CLI

Des kits de développement logiciel (SDK) sont disponibles pour faciliter l’utilisation des API dans des langages de programmation spécifiques. Certains SDK prennent la forme d’interfaces en ligne de commande (CLI), permettant d’envoyer des requêtes API via des commandes dans votre terminal.

Vous pouvez utiliser les CLI suivantes :

  • OSC CLI, développée par 3DS OUTSCALE. Pour en savoir plus, voir Installer et configurer OSC CLI.

  • oapi-cli, développée par 3DS OUTSCALE. Pour en savoir plus, voir Installer et configurer oapi-cli.

  • AWS CLI, développée par Amazon Web Services. Pour en savoir plus, voir Installer et configurer AWS CLI.

Pour en savoir plus sur d’autres SDK disponibles, voir Outils avancés du Cloud.

Interface graphique

Une interface graphique offre un moyen convivial d’interagir avec le Cloud OUTSCALE à l’aide d’éléments visuels tels que des boutons et des menus. Toutefois, les actions disponibles via API ne sont pas toutes disponibles via une interface graphique.

Cockpit est l’interface graphique du Cloud OUTSCALE. C’est une interface web intuitive servant à gérer vos ressources. Pour en savoir plus, voir À propos de Cockpit.

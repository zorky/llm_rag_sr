Créer et publier un produit infogéré

En tant que vendeur partenaire sur OUTSCALE Marketplace, vous pouvez créer un produit infogéré et le publier sur la Marketplace pour que les acheteurs puissent le consulter et y souscrire.

Vous avez accès au tableau Mes produits listant vos produits existants et leur ID produit unique. Vos produits sont publiés, dépubliés, ou à l’état brouillon, c’est-à-dire avec des informations complétées, mais pas encore publiées.

  • Vous devez compléter et enregistrer toutes les sections, dans l’ordre de votre choix, pour pouvoir publier le produit sur la Marketplace. Les sections à compléter sont marquées par un tag .

  • Lorsque vous créez un produit, vous devez fournir toutes les informations et tous les documents en anglais et en français dans les onglets donnés pour pouvoir publier votre produit sur la Marketplace.

Sélection du type de produit

  1. Dans le coin supérieur droit de l’interface, cliquez sur votre nom d’utilisateur ou votre adresse e-mail.
    Un menu déroulant apparaît.

  2. Cliquez sur Mes produits.
    La page Mes produits apparaît.

  3. Cliquez sur Créer un produit.
    La page Type de produit apparaît.

  4. Sélectionnez Produit infogéré.
    La page de Création de votre produit apparaît, avec différentes sections à compléter.

Informations générales

  1. Dans la page Création de votre produit, cliquez sur Accéder dans la section Informations générales.
    Le formulaire Informations générales apparaît.

  2. Importez un logo pour votre produit.

    • Ce logo doit être inférieur à 2 Mo.

    • Nous recommandons d’importer un logo transparent et sans bordures pour un affichage correct.

  3. Saisissez un titre pour votre produit.

  4. Sélectionnez une ou plusieurs catégories pour votre produit.

    Vous pouvez sélectionner jusqu’à trois catégories pour votre produit. Les catégories sélectionnées seront utilisées pour filtrer votre produit.

  5. Saisissez un lien de souscription vers lequel les acheteurs seront redirigés pour souscrire.

  6. Dans la section Synthèse :

    1. Saisissez une description courte pour votre produit.

      Ce champ doit contenir entre 1 et 200 caractères.

    2. Saisissez une description longue pour votre produit.

      Ce champ doit contenir entre 1 et 4000 caractères.

    3. Saisissez les points clés de votre produit.

      Ce champ doit contenir entre 1 et 4000 caractères.

  7. (optionnel) Dans la section Médias :

    1. Importez les fichiers JPG souhaités pour illustrer votre produit.

      • Vous pouvez ajouter jusqu’à 6 fichiers JPG inférieurs à 2 Mo.

      • Le format recommandé est 1440 pixels par 1024 pixels.

    2. Saisissez l’URL d’une vidéo YouTube.

  8. Dans la section Support :

    1. Saisissez les informations concernant le support de votre produit.

    2. Saisissez la politique de mise à jour de votre produit.

  9. Cliquez sur Enregistrer.
    Le formulaire est enregistré et un tag est affiché dans la section Informations générales.

  10. Si vous avez complété toutes les sections du produit et que vous êtes prêt à le publier :

    Dans la page Création de votre produit, cliquez sur Publier le produit pour publier votre produit.
    Votre produit est publié sur la Marketplace et peut être consulté et souscrit par des acheteurs. 3DS OUTSCALE est notifié de la création du produit.

Conditions contractuelles

  1. Dans la page Création de votre produit, cliquez sur Accéder dans la section Conditions contractuelles.
    Le formulaire Conditions contractuelles apparaît.

  2. Prenez connaissance du contrat d’apporteur d’affaires et acceptez le en cochant la case correspondante.

    Une fois le contrat accepté, vous ne pouvez pas modifier votre choix.

  3. Cliquez sur Enregistrer.
    Le formulaire est enregistré et un tag est affiché dans la section Conditions contractuelles.

  4. Si vous avez complété toutes les sections du produit et que vous êtes prêt à le publier :

    Dans la page Création de votre produit, cliquez sur Publier le produit pour publier votre produit.
    Votre produit est publié sur la Marketplace et peut être consulté et souscrit par des acheteurs. 3DS OUTSCALE est notifié de la création du produit.

Glossaire de facturation

Vous trouverez sur cette page un glossaire des intitulés de facturation OUTSCALE, ainsi que des informations sur nos principes de tarification.

Les ressources OUTSCALE sont facturées à la carte ("pay-as-you-go") avec une précision à la seconde : vous payez uniquement les ressources utilisées, et toute heure entamée est facturée à la seconde et non comme une heure entière. Vous pouvez suivre votre consommation à tout moment. Pour en savoir plus, voir Obtenir des informations sur la consommation de vos ressources.

La facturation s’applique aux ressources de calcul, stockage et réseau. Les taux peuvent varier selon la Région et la génération de matériel. Pour en savoir plus, voir les grilles tarifaires sur le site OUTSCALE, ou contacter sales@outscale.com.

Les intitulés de facturation sont au format Nom_produit - Offre - Unité et durée. Les ressources non listées dans ce glossaire, telles que les Nets, ne sont pas facturées.

BSU

BSU - Stockage Magnetic - par Gio et par mois

La capacité de stockage (en Gio) provisionnée lors de la création d’un volume standard, par mois.

Le volume apparaît dans votre compte dans l’un des états suivants :

  • in-use (attaché à une machine virtuelle (VM))

  • available (non attaché à une VM)

Ce taux inclut les IOPS (opérations de lecture et d’écriture par seconde).

BSU - Stockage Performance - par Gio et par mois

La capacité de stockage (en Gio) provisionnée lors de la création d’un volume gp2, par mois.

Le volume apparaît dans votre compte dans l’un des états suivants :

  • in-use (attaché à une VM)

  • available (non attaché à une VM)

Ce taux inclut les IOPS (opérations de lecture et d’écriture par seconde).

BSU - Stockage Enterprise - par Gio et par mois

La capacité de stockage (en Gio) provisionnée lors de la création d’un volume io1, par mois.

Le volume apparaît dans votre compte dans l’un des états suivants :

  • in-use (attaché à une VM)

  • available (non attaché à une VM)

Ce taux n’inclut pas les IOPS (opérations de lecture et d’écriture par seconde). Pour en savoir plus, voir l’entrée [BSU - Stockage Enterprise - par IOPS et par mois] ci-dessous.

BSU - Stockage Enterprise - par IOPS et par mois

Le nombre d’opérations de lecture et d’écriture par seconde (IOPS) provisionné lors de la création d’un volume io1, par mois.

Le volume apparaît dans votre compte dans l’un des états suivants :

  • in-use (attaché à une VM)

  • available (non attaché à une VM)

Ce taux n’inclut pas la capacité de stockage. Pour en savoir plus, voir l’entrée [BSU - Stockage Enterprise - par Gio et par mois] ci-dessus.

DirectLink

DirectLink - Utilisation port 1 Gbps / 10 Gbps - par mois

L’utilisation d’un port DirectLink, par mois.

Si vous utilisez plusieurs ports pour des besoins de redondance, ce taux est multiplié par le nombre de ports.

Le trafic est gratuit pour toutes les Régions où le service est disponible. Pour en savoir plus, voir la grille tarifaire DirectLink.

FCU

FCU transfert de données - Trafic sortant - par Gio

Le volume de données (en Gio) transférées depuis vos VM vers Internet.

Le trafic sortant (depuis vos VM vers Internet) est gratuit pour les Régions suivantes :

  • eu-west-2

  • us-east-2

  • us-west-1

  • cloudgouv-eu-west-1

Le trafic entrant (depuis Internet vers vos VM) est gratuit pour toutes les Régions.

IP publique

EIP - IP externe non associée à une instance démarrée - par heure

L’allocation (réservation) dans votre compte d’une IP publique, par mois.

L’IP publique n’est pas associée (pas utilisée) avec une VM démarrée ou une interface réseau dans le Cloud public.

EIP - IP externe dans un VPC non associée à une instance démarrée - par heure

L’allocation (réservation) dans votre compte d’une IP publique, par mois.

L’IP publique n’est pas associée (pas utilisée) avec une VM démarrée ou une interface réseau dans un Net.

EIP - IP Externe additionnelle dans un VPC associée à une instance démarrée - par heure

Une IP publique supplémentaire associée (utilisée) avec une VM ou une interface réseau dans un Net, par heure.

La première IP publique que vous associez dans chaque Net est gratuite. Ce taux s’applique à partir de la deuxième IP publique associée.

LBU

LBU - Utilisation - par service et par heure

L’utilisation d’un load balancer déployé dans le Cloud public ou dans un Net, par heure.

Ce taux n’inclut pas les données transférées via le load balancer. Pour en savoir plus, voir l’entrée [LBU - Trafic sortant global - par Gio] ci-dessous.

LBU - Trafic sortant global - par Gio

Le volume de données (en Gio) transférées via le load balancer vers Internet.

Le trafic sortant (du load balancer vers Internet) est gratuit pour les Régions suivantes :

  • eu-west-2

  • us-east-2

  • us-west-1

  • cloudgouv-eu-west-1

Le trafic entrant (d’Internet vers le load balancer) est gratuit pour toutes les Régions.

NAT Gateway

NAT Gateway - Utilisation - par heure

L’utilisation d’un NAT service déployé, par heure.

Le NAT service apparaît dans votre compte dans l’état available.

Le trafic entrant (depuis Internet vers le NAT service) est gratuit pour toutes les Régions.

Les ressources supprimées peuvent apparaître dans votre compte pendant une heure, mais ne sont plus facturées.

OOS

OOS - Stockage Enterprise - par Gio et par mois

Le volume de données (en Gio) stockées dans vos buckets OOS, par mois.

Les taux des stockages varient selon la Région. Pour en savoir plus, voir la grille tarifaire stockage objet.

OOS - Trafic sortant - par Gio

Le volume de données (en Gio) transférées depuis vos buckets OOS vers Internet.

Le trafic sortant (de vos buckets OOS vers Internet) est gratuit pour les Régions suivantes :

  • eu-west-2

  • us-east-2

  • us-west-1

  • cloudgouv-eu-west-1

Le trafic entrant (d’Internet vers vos buckets OOS) est gratuit pour toutes les Régions.

Snapshot

Snapshot - Stockage - par Gio et par mois

Premier snapshot d’un volume : La taille (en Gio) de votre snapshot stocké, par mois.

Snapshots suivants du même volume : La taille (en Gio) des données modifiées depuis le snapshot précédent, par mois.

  • Ce taux s’applique à tous les snapshots, y compris ceux créés automatiquement lorsque vous créez une OUTSCALE machine image (OMI) depuis une VM. Pour en savoir plus, voir À propos des OMI.

  • Copier un snapshot entre différentes Régions entraîne des frais supplémentaires pour le transfert des données via Internet. Si le snapshot copié est stocké dans OUTSCALE Object Storage (OOS), les taux de stockage dans les Régions source et destination peuvent également être différents. Pour en savoir plus, voir la section OOS ci-dessus.

VM

Instance - On demand - xx.taille Linux / Windows - par heure

L’utilisation d’une VM, prédéfinie ou personnalisable, par heure.

Ce taux inclut tous les composants de la VM : calcul, mémoire et licences requises. Pour en savoir plus, voir Obtenir le prix de vos ressources.

Ce taux s’applique aux VM à l’état running uniquement. Les VM arrêtées et terminées ne sont pas facturées. Pour en savoir plus, voir À propos du cycle de vie des VM.

Pour en savoir plus sur les types de VM disponibles et leurs capacités, voir Types de VM.

  • Les VM entraînent des frais supplémentaires pour le transfert des données. Pour en savoir plus, voir la section FCU ci-dessus.

  • Un supplément de 10 % s’ajoute au tarif horaire standard des VM dédiées. Pour en savoir plus, voir Obtenir le prix de vos ressources > Calculer le prix des VM dédiées.

Les VM terminées peuvent apparaître dans votre compte pendant une heure, mais ne sont plus facturées.

Instance - Instance dédiée - Prix additionnel - par heure

Un supplément de 2 € pour l’utilisation de VM dédiées.

Ce supplément s’applique une fois par heure lorsque vous avez au moins une VM démarrée dans la Région. Pour en savoir plus, voir Obtenir le prix de vos ressources > Calculer le prix des VM dédiées.

VPN

VPN - Utilisation - par heure

L’utilisation d’une connexion VPN déployée, par heure.

La connexion VPN apparaît dans votre compte dans l’état available.

Le trafic entrant (depuis Internet vers la connexion VPN) est gratuit pour toutes les Régions.

Les ressources supprimées peuvent apparaître dans votre compte pendant une heure, mais ne sont plus facturées.

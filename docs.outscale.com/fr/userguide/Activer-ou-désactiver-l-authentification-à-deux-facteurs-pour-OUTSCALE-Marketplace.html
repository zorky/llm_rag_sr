Activer ou désactiver l’authentification à deux facteurs pour OUTSCALE Marketplace

Lorsque vous avez un compte Marketplace, vous pouvez activer l’authentification à deux facteurs pour augmenter la sécurité de votre compte en exigeant un mot de passe à usage unique ("one-time password" ou "OTP" en anglais) issu d’une application d’OTP. Vous pouvez désactiver l’authentification à deux facteurs pour votre compte à tout moment.

Pour en savoir plus, voir À propos d’OUTSCALE Marketplace > Création d’une organisation et connexion.

Activer l’authentification à deux facteurs

Avant de commencer : Installez une application d’OTP sur votre smartphone ou sur un autre appareil, de préférence un appareil différent que celui utilisé pour naviguer sur la Marketplace.

Paramétrer un appareil d’authentification

Vous pouvez paramétrer autant d’appareils d’authentification que nécessaire.

  1. Dans le coin supérieur droit de l’interface, cliquez sur votre nom d’utilisateur ou votre adresse e-mail.
    Un menu déroulant apparaît.

  2. Cliquez sur Paramètres d’authentification.
    Une page de gestion de l’authentification apparaît.

  3. Dans la section Two-factor authentication, cliquez sur Set up Authenticator application.
    La page Configuration de l’authentification par mobile apparaît.

  4. Ouvrez l’application d’OTP sur votre appareil et scannez le QR code affiché.

    Si vous ne parvenez pas à scanner le QR code, cliquez sur Impossible de scanner? pour afficher une clé à saisir manuellement dans votre application d’OTP.

  5. Dans le champ Code à usage unique, saisissez le code à usage unique fourni par l’application.

  6. Dans le champ Device Name, saisissez un nom pour l’appareil correspondant.

  7. Cliquez sur Soumettre.
    L’authentification à deux facteurs est activée pour votre compte. Vous pouvez vous connecter à la Marketplace en saisissant un OTP généré par l’appareil correspondant.

Désactiver l’authentification à deux facteurs

Supprimer un appareil d’authentification

Pour complètement désactiver l’authentification à deux facteurs pour votre compte, vous devez supprimer tous les appareils d’authentification.

  1. Dans le coin supérieur droit de l’interface, cliquez sur votre nom d’utilisateur ou votre adresse e-mail.
    Un menu déroulant apparaît.

  2. Cliquez sur Paramètres d’authentification.
    Une page de gestion de l’authentification apparaît.

  3. Dans la section Two-factor authentication, cliquez sur Remove sur la ligne de l’appareil que vous voulez supprimer.
    Une boîte de dialogue de confirmation apparaît.

  4. Cliquez sur Continue.
    L’appareil d’authentification est supprimé et vous ne pouvez plus vous connecter en utilisant cet appareil.

Souscrire à un produit infogéré

En tant qu’acheteur sur OUTSCALE Marketplace, vous pouvez souscrire à un produit infogéré depuis le catalogue de la Marketplace.

Les vendeurs partenaires peuvent dépublier des produits infogérés et ont la responsabilité d’en informer les souscripteurs et les administrateurs de leurs organisations.

Pour en savoir plus sur les produits infogérés, voir À propos d’OUTSCALE Marketplace > Produits infogérés.

Pour souscrire à un produit infogéré :

  1. Dans l’interface de la Marketplace, cliquez sur l’onglet Catalogue.
    Le catalogue de produits apparaît.

  2. Utilisez les filtres sur la gauche pour trouver le produit souhaité.

  3. Cliquez sur la vignette produit souhaitée.
    La page du produit apparaît.

  4. Cliquez sur Contacter le vendeur partenaire.
    Un formulaire partiellement prérempli apparaît.

    Le type de requête sélectionné par défaut est Demande de devis, mais cette valeur peut être modifiée.

  5. (optionnel) Modifiez les informations préremplies si nécessaire.

  6. Spécifiez les informations requises manquantes.

  7. Cliquez sur Envoyer la demande.
    La requête est envoyée au vendeur partenaire par e-mail.

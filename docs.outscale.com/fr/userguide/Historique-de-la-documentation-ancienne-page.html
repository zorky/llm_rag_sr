Historique de la documentation (ancienne page)

Cette page liste les changements majeurs de la documentation publique. Les changements plus mineurs ne sont pas mentionnés.

2024

17 juin 2024

Depuis le 17 juin 2024, les notes de mise à jour se trouvent sur une nouvelle page : Notes de mise à jour.

23 mai 2024

  • Un guide technique expliquant comment utiliser la pagination des requêtes API est maintenant disponible : Tutoriel : Paginer une requête API.

26 avril 2024

  • La page Matrice de compatibilité avec AWS est plus exhaustive :

    • Auparavant, cette page listait les correspondances de méthodes d’API uniquement dans le sens AWS → OUTSCALE, ce qui signifie que les méthodes de l’API OUTSCALE n’ayant pas d’équivalents AWS n’étaient pas mentionnées. Désormais, cette page mentionne toutes les méthodes y compris celles exclusives à l’API OUTSCALE.

    • En outre, vous pouvez désormais trier le tableau en cliquant sur les en-têtes des colonnes.

  • Le Générateur de politiques EIM offre désormais un deuxième format de sortie :

    • En plus du format JSON brut, utilisable par exemple avec Cockpit, la politique générée est également présentée sous la forme d’une chaîne JSON, qui est la forme nécessaire si vous voulez coller ce résultat dans le paramètre d’une commande OSC CLI.

14 mars 2024

  • Le thème visuel du site web de documentation a été mis à jour, afin d’être en harmonie avec les couleurs des autres sites web OUTSCALE.

19 février 2024

  • Les conditions Wait sont à présent documentées dans la section OUTSCALE Object Storage (OOS) : Utiliser les conditions Wait.

23 janvier 2024

  • Une page interactive est désormais disponible pour vous aider à générer des politiques EIM : Générateur de politiques EIM.

15 janvier 2024

  • Un guide technique expliquant comment configurer et utiliser la fonctionnalité Discard est maintenant disponible : Configurer et utiliser le Discard.

2023

29 décembre 2023

  • Fin de vie (EOL) de Cockpit v1. À compter de cette date, Cockpit v1 n’est plus disponible sur aucune de nos Régions. Pour en savoir plus, voir Politique de fin de vie.

30 novembre 2023

  • La fonctionnalité Object Lock est à présent disponible dans la documentation OUTSCALE Object Storage (OOS). Pour en savoir plus, voir Gérer Object Lock.

  • À compter de cette date, Cockpit v1 n’est plus disponible sur la Région cloudgouv-eu-west-1. Pour en savoir plus, voir Politique de fin de vie.

29 novembre 2023

  • La page documentant les codes d’erreurs de l’API OUTSCALE a été mise à jour. Pour en savoir plus, voir Liste des erreurs de l’API OUTSCALE.

27 novembre 2023

  • La politique de mot de passe a été mise à jour. Pour en savoir plus, voir À propos de la protection par mot de passe.

20 octobre 2023

  • Fin de support et maintenance (EOSM) de Cockpit v1. À compter de cette date, Cockpit v1 n’est plus supporté ni corrigé. Pour en savoir plus, voir Politique de fin de vie.

5 octobre 2023

  • Dans le Guide utilisateur, les pages de procédures incluent désormais des descriptions et des exemples pour OSC CLI, au lieu d’uniquement renvoyer vers des sections de la documentation OUTSCALE API. Par exemple, voir Créer des VM > Créer des VM avec OSC CLI.

  • Ces descriptions et exemples d’OSC CLI (actuellement uniquement en anglais) sont les mêmes entre le Guide utilisateur et la documentation OUTSCALE API. Vous pouvez donc choisir d’utiliser l’une ou l’autre partie du site en fonction de votre préférence de navigation.

20 septembre 2023

  • Cockpit v2 sort de bêta suite au déploiement de la version 2.9.

    • En conséquence, le Guide utilisateur utilisera principalement désormais les termes employés par Cockpit v2, qui sont aussi ceux de l’API OUTSCALE (par exemple, les ressources sont appelées machines virtuelles, Nets, etc.), plutôt que la terminologie historique AWS qui était précédemment employée (par exemple, instances, VPC, etc.). Les infobulles restent disponibles au survol de la souris pour faciliter la transition. Ce changement de terminologie s’applique aussi aux titres des pages (par exemple, À propos des instances redirige désormais vers À propos des VM).

    • Pour en savoir plus sur la version 2.9 de Cockpit v2, voir les Notes de mise à jour.

18 septembre 2023

  • Une nouvelle version de IAM est sortie. Une page résumant toutes les nouvelles fonctionnalités et mises à jour est disponible : Référence des fonctionnalités IAM 1.5.

  • Par conséquent, vous pouvez trouver les nouveaux contenus suivants dans la documentation :

    • À propos de l’authentification

    • Authentification multifacteur

    • Activer une session de confiance

    • Créer un mot de passe en tant qu’utilisateur EIM

30 août 2023

  • La documentation de l’API OUTSCALE inclut désormais des exemples de code pleinement fonctionnels :

    • Chacune des 180 méthodes disponibles comporte au moins un exemple de code par langage de programmation.

    • Certaines des méthodes comportent plusieurs exemples de code par langage de programmation. Par exemple, la méthode CreateImage comporte quatre exemples différents, chacun pour un cas d’usage différent au niveau des paramètres de requête.

    • Par commodité, deux boutons copier s’affichent à côté de chaque exemple de code au survol de la souris :

      •  : Le premier bouton copie le code tel quel, donc sous forme de chaîne multiligne.

      •  : Le second bouton copie le code sous la forme d’une commande monoligne (lorsque possible).

27 juillet 2023

  • La fonctionnalité de recherche du Guide utilisateur a été améliorée :

    • Lorsque vous utilisez la barre de recherche en haut à droite, vous pouvez désormais consulter les résultats dans une page distincte et non plus seulement dans la liste déroulante.

    • Vous pouvez désormais ajouter la documentation publique aux moteurs de recherche de votre navigateur. Cela vous permet de rechercher rapidement dans le site de documentation à l’aide d’un raccourci/mot-clé. Par exemple, vous pouvez taper osc vm types dans votre barre d’URL pour lancer directement une recherche pour "vm types" dans la documentation, même si vous n’êtes pas déjà sur le site. Pour en savoir plus, voir Rechercher dans la documentation publique depuis votre barre d’URL.

  • Vous pouvez désormais exporter au format PDF toute page du Guide utilisateur, en utilisant le bouton situé dans la colonne de droite de chaque page.

27 juin 2023

  • Les documentations API ont été mises à jour :

    • Les sections Authentication ont été réécrites.

    • Des exemples de code pour l’authentification par login/mot de passe sont désormais inclus.
      Notez que l’authentification par login/mot de passe est supportée uniquement pour certaines actions, dans l’API OUTSCALE (par exemple, ReadAccessKeys) et dans l’API ICU (par exemple, ListAccessKeys).

    • Des exemples de code en JavaScript sont désormais inclus pour l’API OUTSCALE.
      Ils sont basés sur le SDK OUTSCALE pour TypeScript/JavaScript.

    • Vous pouvez désormais basculer entre les langages de programmation en utilisant les boutons au-dessus de chaque exemple de code, ainsi que dans le coin supérieur gauche.

23 juin 2023

  • La Matrice de compatibilité avec AWS a été mise à jour. Cette page liste les correspondances entre méthodes API AWS-compliant et méthodes API OUTSCALE.

3 mai 2023

  • Des flux RSS sont désormais disponibles pour vous aider à suivre les mises à jour de la documentation. Vous pouvez vous abonner aux flux suivants, qui correspondent aux trois onglets de la page d’accueil :

    • Mises à jour majeures de la documentation

    • Notes de releases Cockpit v2

    • Notes de releases API

23 février 2023

  • Les documentations API incluent désormais une barre de recherche dans le coin supérieur gauche.

31 janvier 2023

  • Une page décrivant les différents niveaux de maturité des projets open source OUTSCALE est désormais disponible : Projets open source.

2022

20 décembre 2022

  • Deux guides techniques concernant cloud-init sont désormais disponibles :

    • L’un constitue une introduction générale à l’outil cloud-init : Introduction à cloud-init.

    • L’autre présente une approche pas à pas pour utiliser cloud-init dans une VM OUTSCALE : Utiliser cloud-init avec les user data.

16 novembre 2022

  • Un guide technique expliquant comment activer la virtualisation imbriquée est désormais disponible : Activer la virtualisation imbriquée.

4 octobre 2022

  • Toutes les procédures du Guide utilisateur pouvant être exécutées avec OSC CLI incluent désormais des liens vers les sections correspondantes de la documentation API OUTSCALE.

20 septembre 2022

  • Une page résumant les concepts et la compatibilité du service Elastic Identity Management (EIM) est désormais disponible : À propos d’EIM.

22 août 2022

  • Publication de la page d’historique.

19 août 2022

  • Pour améliorer l’expérience utilisateur des documentations API, les bannières rouges et vertes indiquant les méthodes d’authentification ont été déplacées vers le début de chaque section de méthode API.

16 août 2022

  • En termes de gouvernance pour votre compte, une bonne pratique peut être de créer un utilisateur EIM dont le seul droit est de lire vos informations de consommation. Ainsi, un nouveau tutoriel est disponible : Tutoriel : Obtenir votre consommation avec un utilisateur EIM.

12 août 2022

  • L’affichage des descriptions de paramètres multilignes a été corrigé dans les documentations API.

13 juillet 2022

  • Dans les documentations API, les exemples de code ont été mis à jour :

    • Pour l’API OUTSCALE : Les exemples existants avec OSC CLI, Curl, et en Python ont été améliorés.

    • Pour les API FCU, LBU, EIM, ICU, et DirectLink : Des exemples avec OSC CLI et Curl sont désormais inclus.

23 juin 2022

  • La documentation de Cockpit v2-beta est désormais en ligne. Pour en savoir plus sur Cockpit v2, voir À propos de Cockpit > Cockpit v2 et pour en savoir plus sur les fonctionnalités disponibles, voir Référence des fonctionnalités Cockpit v2.

23 mai 2022

  • Une page listant les erreurs de l’API OUTSCALE est désormais disponible (en anglais uniquement). Pour en savoir plus, voir Liste des erreurs de l’API OUTSCALE.

3 mai 2022

  • Les FAQ ont été mises à jour et sont désormais disponibles sur le site de notre Support. Pour en savoir plus, voir https://support.outscale.com.

14 mars 2022

  • Un nouveau tutoriel pour OUTSCALE Monitoring Services (OMS) est désormais disponible : Tutoriel : Archiver vos logs d’API.

18 février 2022

  • La documentation OUTSCALE Object Storage (OOS) a été mise à jour. Des procédures détaillées sont désormais disponibles à propos du partage de ressources cross-origin (CORS) pour les buckets, du cycle de vie des objets, et des tags des objets. Pour en savoir plus, voir Gérer CORS sur vos buckets, Gérer le cycle de vie de vos buckets, et Gérer les tags d’un objet.

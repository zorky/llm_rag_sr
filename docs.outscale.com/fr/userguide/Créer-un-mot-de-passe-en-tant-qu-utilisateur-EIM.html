Créer un mot de passe en tant qu’utilisateur EIM

En tant qu’utilisateur EIM, vous pouvez créer un mot de passe que vous pouvez ensuite utiliser pour vous connecter à Cockpit v2.

Cette méthode est uniquement disponible sur Cockpit v2.

Créer un mot de passe en tant qu’utilisateur EIM sur Cockpit v2

Avant de commencer : Assurez-vous qu’une adresse e-mail a été assignée à votre compte EIM user. Pour en savoir plus, voir Créer un utilisateur et Modifier un utilisateur.

  1. Accédez à Cockpit v2.

  2. Sélectionnez la Région où se trouve votre compte OUTSCALE.

  3. Dans le champ Nom d’utilisateur, entrez votre identifiant.

    Votre identifiant suit le format suivant : ID du compte racine/votre nom d’utilisateur (par exemple : 012345678912/username).

  4. Cliquez sur Se connecter.
    Le champ Mot de passe apparaît.

  5. Cliquez sur Mot de passe oublié ?.
    La page Mot de passe oublié apparaît.

  6. Dans le champ Nom d’utilisateur ou e-mail, entrez votre identifiant et cliquez sur Envoyer le lien.

    Si aucune adresse e-mail ne vous a été assignée, le lien est envoyé à l’adresse e-mail du compte racine.

  7. Cliquez sur le lien dans l’e-mail de réinitialisation de mot de passe reçu.
    La page Réinitialiser le mot de passe apparaît.

  8. Dans le champ Nouveau mot de passe, entrez votre nouveau mot de passe.

  9. Dans le champ Confirmation du nouveau mot de passe, entrez votre nouveau mot de passe une seconde fois.

  10. Cliquez sur Réinitialiser.
    Votre mot de passe est créé. Vous pouvez désormais l’utiliser pour vous connecter à Cockpit v2 en tant qu’utilisateur EIM.

Lister vos buckets

Vous pouvez lister les buckets que vous avez créés. Cette action renvoie également des informations à propos de la date et l’heure de leur création.

Les buckets que les autres utilisateurs ont partagés avec vous n’apparaissent pas.

Lister vos buckets avec AWS CLI (s3api)

Avant de commencer : Installez et configurez AWS CLI. Pour en savoir plus, voir Installer et configurer AWS CLI.

Pour obtenir la liste de vos buckets, utilisez la commande list-buckets en suivant cette syntaxe :

Exemple de requête
$ aws s3api list-buckets \
    --profile YOUR_PROFILE \
    --endpoint https://oos.eu-west-2.outscale.com

Cette commande contient les attributs suivants que vous devez spécifier :

  • (optionnel) profile : Le profil nommé que vous voulez utiliser, créé pendant la configuration d’AWS CLI. Pour en savoir plus, voir Installer et configurer AWS CLI.

  • endpoint : Le endpoint correspondant à la Région à laquelle vous voulez envoyer la requête. Pour en savoir plus, voir Installer et configurer AWS CLI.

La commande list-buckets renvoie les éléments suivants :

  • Owner : Informations à propos du propriétaire du bucket. Cet élément contient les informations suivantes :

    • DisplayName : L’ID de votre compte OUTSCALE.

    • ID : Votre ID utilisateur S3.

  • Buckets : Informations à propos de vos buckets. Cet élément contient les informations suivantes pour chaque bucket :

    • Name : Le nom que vous avez spécifié pour le bucket.

    • CreationDate : La date et l’heure de la création du bucket, au format ISO 8601.

Exemple de résultat
{

    "Owner": {
        "DisplayName": "ACCOUNT_ID",
        "ID": "USER_ID"
    },
    "Buckets": [
        {
            "CreationDate": "2017-01-01T08:03:16.000Z",
            "Name": "BUCKET"
            "Name": "bucket1",
            "CreationDate": "2023-11-29T08:51:46.760000+00:00"
        },
        {
            "Name": "bucket2",
            "CreationDate": "2024-01-31T14:47:33.303000+00:00"
        },
        {
            "Name": "bucket3",
            "CreationDate": "2024-01-30T14:59:21.190000+00:00"
        }
    ]
}

Lister vos buckets avec AWS CLI (s3)

Avant de commencer : Installez et configurez AWS CLI. Pour en savoir plus, voir Installer et configurer AWS CLI.

Pour obtenir la liste de vos buckets, utilisez la commande ls en suivant cette syntaxe :

$ aws s3 ls \
    --profile YOUR_PROFILE \
    --debug \
    --endpoint https://oos.eu-west-2.outscale.com

Cette commande contient les attributs suivants que vous devez spécifier :

  • (optionnel) profile : Le profil nommé que vous voulez utiliser, créé pendant la configuration d’AWS CLI. Pour en savoir plus, voir Installer et configurer AWS CLI.

  • (optionnel) debug : Si inclus, renvoie un rapport détaillé de l’opération. Cet attribut est utile pour analyser et résoudre les problèmes que vous pouvez rencontrer.

  • endpoint : Le endpoint correspondant à la Région à laquelle vous voulez envoyer la requête. Pour en savoir plus, voir Installer et configurer AWS CLI.

La commande ls renvoie une liste des buckets ainsi que la date et l’heure de leur création, en heure locale.

Result Sample
2023-11-29 09:51:46 bucket1
2024-01-30 10:30:46 bucket2
2024-01-30 15:59:21 bucket3

Lister vos buckets avec s3cmd

Vous pouvez gérer vos ressources de stockage objet en utilisant des commandes s3cmd. Pour en savoir plus, voir s3cmd.

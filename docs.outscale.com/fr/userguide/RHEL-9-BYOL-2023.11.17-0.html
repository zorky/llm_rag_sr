Enregistrement d’une license

Enregistrement au premier démarrage

Au premier démarrage d’une machine virtuelle (VM) avec ce système d’exploitation Red Hat Enterprise Linux, vous devez enregistrer votre propre licence. Pour en savoir plus sur la procédure, voir le site officiel de Red Hat.

Vous pouvez également utiliser la commande suivante :

$ subscription-manager register \
    --username <username> \
    --password <password> \
    --auto-attach

Enregistrement via Cloud Init

Vous pouvez utiliser Cloud Init pour configurer votre système pendant l’installation et le premier démarrage, puis enregistrer votre propre licence. Pour en savoir plus sur la procédure, voir le site officiel de Red Hat.

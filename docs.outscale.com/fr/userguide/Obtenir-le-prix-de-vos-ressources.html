Obtenir le prix de vos ressources

Vous pouvez obtenir des informations sur le prix de toute ressource OUTSCALE dans la Région de votre compte sur Cockpit v2. Notez toutefois que ces informations sont fournies à titre indicatif uniquement. Seules les factures ont valeur contractuelle.

Vous pouvez aussi les obtenir par le biais de la méthode ReadCatalog. Pour en savoir plus, voir les méthodes API correspondantes.

Pour calculer le prix de machines virtuelles (VM) personnalisables ou dédiées, vous devez obtenir différents prix listés dans le résultat de la méthode API ReadCatalog, puis utiliser une formule pour calculer le prix final selon les caractéristiques de votre ressource.

Vous pouvez obtenir la liste des prix des services et ressources OUTSCALE dans une période déterminée (jusqu’à 3 ans) avec la méthode API OUTSCALE ReadCatalogs. Pour en savoir plus, voir l’exemple de commande ReadCatalogs dans la documentation de l’API OUTSCALE.

Si vous n’avez pas d’identifiants OUTSCALE, vous pouvez utiliser la méthode API ReadPublicCatalog pour obtenir le prix des ressources OUTSCALE pour la Région spécifiée dans le endpoint de votre requête. Pour en savoir plus, voir À propos des Régions et Sous-régions.

Obtenir le prix de vos ressources avec Cockpit v2

  1. Dans le coin supérieur droit, cliquez sur l’icône .
    Un menu déroulant apparaît.

  2. Cliquez sur  Espace personnel.
    Les informations de votre compte apparaissent.

  3. Cliquez sur l’onglet Ma consommation.
    La consommation de votre compte depuis le premier jour du mois en cours apparaît.

  4. Le prix de vos ressources est disponible dans le tableau, sous les colonnes Prix unitaire et Prix total.

    Le prix est affiché hors taxes.

Calculer le prix des VM personnalisables

Obtenez les trois prix suivants dans le catalogue :

  • Le prix d’un vCore, par heure (Instance - On demand - vCore unit for a CxRy Linux/Windows/Oracle instance - per hour).

  • Le prix pour 1 Gio de mémoire, par heure (Instance - On demand - RAM unit for a CxRy Linux/Windows/Oracle instance - per hour).

  • Le prix produit pour les VM personnalisables, par heure (Instance - On demand - Additional price for a CxRy Linux/Windows/Oracle instance - per hour).

Ces trois prix dépendent du type de produit de votre VM (Linux, Windows, ou Oracle). Pour en savoir plus sur les VM personnalisables Linux et Windows, voir Types de VM.

Calculer le prix des VM dédiées

Obtenez les deux prix suivants dans le catalogue :

  • Le prix d’utilisation pour les VM dédiées, par heure (Instance - Dedicated instances usage - per hour).

  • Le prix additionnel pour les VM dédiées, en pourcentage et par heure (Instance - Dedicated Instance - Additional price - per hour).

Calculez le prix par heure de votre VM dédiée, correspondant au prix de la VM classique majoré par le prix additionnel, à l’aide de la formule suivante :

prix_instance + ((prix_instance x prix_additionnel)/100)

Cette formule utilise les informations suivantes :

  • prix_instance : Le prix de la VM, par heure.

    Si votre VM dédiée est une VM personnalisable, vous devez d’abord calculer son prix par heure à l’aide de la méthode correspondante présentée dans la section Calculer le prix des VM personnalisables ci-dessus.

  • prix_additionnel : Le prix additionnel pour les VM dédiées, en pourcentage et par heure.

Pour obtenir le prix total par heure pour votre VM dédiée, ajoutez le prix additionnel pour les VM dédiées au prix par heure de votre VM dédiée calculé à l’étape précédente.

Le prix additionnel est facturé une fois uniquement, du moment où vous créez votre première VM dédiée jusqu’au moment où vous terminez votre dernière VM dédiée.

Le prix facturé pour cet exemple est :

  • Le prix de la VM 3 et son prix additionnel pour les VM dédiées correspondant, pour une heure.

  • Le prix d’utilisation 2 pour les VM dédiées pour une heure (du moment où la VM 3 est créée au moment où elle est terminée).

La devise utilisée pour la facturation dépend de la Région de votre compte :

  • eu-west-2 et cloudgouv-eu-west-1 : EUR

  • us-east-2, us-west-1, et ap-northeast-1 : USD

Calculer le prix des VM réservées

  • Pour obtenir le prix d’une VM réservée, utilisez une des formules suivantes selon la durée choisie pour votre réservation :

    • 1 mois (30 % de réduction) :

      prix_instance x 24 x 30 x 0.7

    • 1 an (40 % de réduction) :

      prix_instance x 24 x 365 x 0.6

    • 2 ans (50 % de réduction) :

      prix_instance x 24 x 365 x 2 x 0.5

    • 3 ans (60 % de réduction) :

      prix_instance x 24 x 365 x 3 x 0.4

    Le prix_instance correspond au prix par heure de votre VM :

    • Les tarifs sont disponibles à l’adresse suivante : https://fr.outscale.com/tarifs/.

    • Si votre VM est une VM personnalisable et/ou une VM dédiée, vous devez d’abord calculer son prix par heure à l’aide des méthodes correspondantes présentées dans les sections Calculer le prix des VM personnalisables et Calculer le prix des VM dédiées ci-dessus.

    La devise utilisée pour la facturation dépend de la Région de votre compte :

    • eu-west-2 et cloudgouv-eu-west-1 : EUR

    • us-east-2, us-west-1, et ap-northeast-1 : USD

À propos des API

Vous pouvez gérer vos ressources du Cloud OUTSCALE avec plusieurs interfaces de programmation applicatives (API).

D’autres outils sont disponibles pour gérer vos ressources, dont l’interface web Cockpit. Pour en savoir plus, voir Outils du Cloud.

API OUTSCALE

3DS OUTSCALE expose son API publique. Cette API est le principal moyen de gérer vos ressources dans le Cloud OUTSCALE, car toutes les fonctionnalités OUTSCALE sont développées d’abord pour cette API.

L’API OUTSCALE est accessible depuis un unique endpoint pour tous les services.

La documentation de l’API OUTSCALE est accessible ici.

Une description OpenAPI est également disponible dans ce dépôt GitHub.

API AWS-compliant

3DS OUTSCALE fournit aussi des API conformes à Amazon Web Services (AWS). Ces API sont accessibles depuis différents endpoints selon le service.

Pour en savoir plus sur la compatibilité des appels entre les API développées par 3DS OUTSCALE et les API développées par AWS, voir Matrice de compatibilité avec les API AWS.

Les documentations des API AWS-compliant sont accessibles via les liens suivants :

  • Flexible Compute Unit (FCU)

  • Load Balancer Unit (LBU)

  • Elastic Identity Management (EIM)

  • Internal Call Unit (ICU)

  • DirectLink

  • OUTSCALE Object Storage (OOS)

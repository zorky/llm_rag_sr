Tutoriel : Mettre en place une connexion DirectLink pour un Net

Le but de ce tutoriel est de créer une connexion DirectLink qui vous permet d’accéder à un Net que vous avez créé dans une Région correspondant à un site DirectLink.

Dans ce tutoriel, vous apprenez :

  • Le process et des prérequis pour créer une connexion DirectLink

  • Comment configurer la connexion DirectLink pour un Net

  1. Identifiez le site DirectLink pour la Région à laquelle vous souhaitez vous connecter. Pour en savoir plus, voir Obtenir des informations sur les sites DirectLink.

  2. Créez le Net et la virtual gateway requis pour une connexion DirectLink :

    1. Créez un Net dans la Région où vous souhaitez créer la connexion DirectLink. Pour en savoir plus, voir Créer un Net.

    2. Créez une virtual gateway dans cette Région. Pour en savoir plus, voir Créer une virtual gateway.

    3. Attachez cette virtual gateway au Net. Pour en savoir plus, voir Attacher une virtual gateway.

  3. Assurez-vous que vous avez un routeur ou une fibre optique appropriés dans le même data center que le site DirectLink, ou demandez à un prestataire de service réseau d’y amener une fibre pour vous. Pour en savoir plus sur les prérequis techniques nécessaires pour votre routeur et le câble utilisé pour la connexion, voir À propos de DirectLink.

  4. Demandez une connexion DirectLink pour le site de votre choix avec la bande passante dont vous avez besoin. Pour en savoir plus, voir Demander une connexion.
    Une fois la connexion acceptée, vous recevez la lettre d’autorisation (Letter of Authorization - LOA) envoyée par 3DS OUTSCALE par e-mail.

  5. Donnez la LOA à votre prestataire de service réseau qui demandera au data center de connecter votre fibre optique au routeur DirectLink.
    La connexion physique est créée.

  6. Configurez votre connexion physique avec une ou plusieurs DirectLink interfaces à l’aide de l’ID de la virtual gateway attachée à votre Net. Pour en savoir plus, voir Créer une DirectLink interface.
    Votre connexion DirectLink est mise en place.

    Vous pouvez créer plusieurs DirectLink interfaces pour une même connexion DirectLink afin d’accéder à plusieurs Nets ou utiliser plusieurs VLAN.

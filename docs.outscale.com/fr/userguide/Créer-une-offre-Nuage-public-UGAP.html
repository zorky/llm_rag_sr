Créer une offre Nuage public (UGAP)

En tant que vendeur partenaire au sein d’une organisation appartenant au marché Nuage public de l’Union des groupements d’achats publics (UGAP), vous pouvez créer une ou plusieurs offres d’abonnement ou anuelles et horaires par type de VM pour un produit image machine OUTSCALE (OMI) à destination d’acheteurs de ce même marché.

  • Votre organisation doit être validée par 3DS OUTSCALE pour pouvoir créer une offre Nuage public. Pour en savoir plus, voir Créer une organisation en tant que vendeur partenaire.

  • Vous devez créer au moins une offre pour tous les acheteurs en plus de la ou des offres pour le marché Nuage public afin de pouvoir publier votre produit dans le catalogue.

Créer des offres est une étape du processus de création et de publication d’un produit OMI. Pour en savoir plus sur le reste du processus, voir Créer et publier un produit OMI.

Offre d’abonnement

Pour créer une offre d’abonnement pour le marché Nuage public :

  1. Dans la page Création de votre produit, cliquez sur Accéder dans la section Offres.
    La page Offres apparaît.

  2. Cliquez sur Créer une offre.
    Le formulaire Offre apparaît.

  3. Dans la section Paramètres de votre offre :

    1. Saisissez un nom pour votre offre.

    2. Saisissez une description pour votre offre.

    3. Sélectionnez la cible Nuage public (UGAP).

  4. Dans la section Tarif de votre offre, définissez un taux hors taxes.

    Le taux ne peut pas être modifié après la publication du produit.

    La durée d’engagement et la fréquence de facturation de votre offre sont déjà définies par défaut.

  5. Dans la section VM recommandées, ajoutez au moins une VM recommandée pour votre produit :

    1. Cliquez sur Ajouter une VM recommandée.

    2. Spécifiez les paramètres de la VM.

    3. Cliquez sur Enregistrer.

      • Vous pouvez modifier les VM recommandées en cliquant sur Modifier pour la VM souhaitée.

      • Vous pouvez supprimer les VM recommandées en cliquant sur Supprimer pour la VM souhaitée.

      Pour en savoir plus sur les types de VM, voir Types de VM.

  6. Cliquez sur Enregistrer.
    Le formulaire est enregistré et un tag est affiché dans la section Offre.

Offre annuelle par type de VM

Pour créer une offre annuelle par type de VM pour le marché Nuage public :

  1. Dans la page Création de votre produit, cliquez sur Accéder dans la section Offres.
    La page Offres apparaît.

  2. Cliquez sur Créer une offre.
    Le formulaire Offre apparaît.

  3. Dans la section Paramètres de votre offre :

    1. Saisissez un nom pour votre offre.

    2. Saisissez une description pour votre offre.

  4. Dans la section Paramètres de votre tarif :

    1. Cliquez sur Ajouter un type de VM.
      La fenêtre Ajouter un type de VM apparaît.

    2. Spécifiez les paramètres de la VM.

    3. Cliquez sur Enregistrer.
      Une ligne est ajoutée dans le tableau avec les informations que vous venez de renseigner.

    4. Cliquez sur le champ Licence/heure pour spécifier un tarif horaire hors taxes pour votre licence, en fonction du type de VM paramétré.

      Lorsque vous créez une offre annuelle par type de VM destinée au marché Nuage public, vous devez renseigner un tarif horaire et annuel pour votre licence, en plus du tarif Nuage public, afin de créer une offre pour tous les acheteurs en parallèle de votre offre Nuage public.

    5. Cliquez sur le champ Licence/an pour spécifier un tarif annuel hors taxes pour votre licence, en fonction du type de VM paramétré.
      Le champ Licence Nuage public/an est prérempli en fonction du tarif annuel que vous venez de rensigner.

    6. Cliquez sur le champ Licence Nuage public/an pour l’ajuster selon vos besoins.

      • Vous pouvez ajouter autant de types de VM que nécessaire.

      • Vous ne pouvez pas ajouter deux fois un même type de VM.

      • Pour en savoir plus sur les types de VM, voir Types de VM.

      • Vous pouvez modifier les types de VM en cliquant sur Modifier pour le type de VM souhaité.

      • Vous pouvez supprimer les types de VM en cliquant sur Supprimer pour le type de VM souhaité.

      Vous ne pouvez pas modifier ou supprimer un type de VM une fois que l’offre est publiée.

    7. Spécifiez un tarif par défaut hors taxes pour la licence.

      Le tarif par défaut est appliqué lorsqu’un acheteur utilise votre produit sur une VM qui n’est pas incluse dans les types de VM de votre offre.

  5. Cliquez sur Enregistrer.
    Le formulaire est enregistré et un tag est affiché dans la section Offre.

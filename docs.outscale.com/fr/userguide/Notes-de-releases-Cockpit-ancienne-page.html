Notes de releases Cockpit v2 (ancienne page)

Cette page liste les nouvelles fonctionnalités incluses dans chaque release de Cockpit v2.

2024

17 juin 2024

Depuis le 17 juin 2024, les notes de mise à jour se trouvent sur une nouvelle page : Notes de mise à jour.

23 avril 2024 (Cockpit 2.16)

  • Les fonctionnalités suivantes sont désormais disponibles dans Cockpit v2 :

    • Groupes EIM :

      • Ajouter ou retirer un utilisateur d’un groupe

      • Obtenir des informations sur vos groupes

    • Politiques EIM

      • Lister des politiques gérées attachées à un groupe

14 mars 2024 (Cockpit 2.15)

  • Les fonctionnalités suivantes sont désormais disponibles dans Cockpit v2 :

    • Logs d’API :

      • Obtenir des informations sur vos logs d’API

    • Consommation :

      • Obtenir le prix de vos ressources

  • La génération de CPU TINA v3 n’est plus disponible lors de la création d’une VM. Pour en savoir plus, voir Politique de fin de vie.

  • Vous pouvez désormais ajouter des tags aux options DHCP. Pour en savoir plus, voir À propos des tags.

  • Une nouvelle charte graphique est disponible sur Cockpit v2.

20 février 2024 (Cockpit 2.14)

  • Les fonctionnalités suivantes sont désormais disponibles dans Cockpit v2 :

    • Règles d’accès API :

      • Créer une règle d’accès API

      • Obtenir des informations sur vos règles d’accès API

      • Modifier une règle d’accès API

      • Supprimer une règle d’accès API

29 janvier 2024 (Cockpit 2.13)

  • Les fonctionnalités suivantes sont désormais disponibles dans Cockpit v2 :

    • Utilisateurs EIM :

      • Modifier une access key EIM

      • Supprimer une access key EIM

    • Politiques EIM :

      • Lister des politiques gérées attachées à un utilisateur

2022

8 décembre 2022 (Cockpit 2.3)

  • Lors de la création d’une client gateway, vous pouvez désormais spécifier son nom.

  • Cette release améliore la stabilité et certains messages d’erreur.

20 septembre 2022 (Cockpit 2.2)

  • La fonctionnalité Créer une connexion VPN est désormais disponible.

  • Dans Espace personnel > Ma consommation, vous pouvez choisir d’afficher la consommation consolidée de votre paying account et de ses comptes liés, ou la consommation individuelle de votre compte. Pour en savoir plus, voir Obtenir des informations sur la consommation de vos ressources.

    • Supprimer un NAT service

20 juin 2022 (Cockpit 2.0)

  • Cockpit v2-beta est disponible. Pour en savoir plus à propos de Cockpit v2, voir À propos de Cockpit > Cockpit v2-beta et pour en savoir plus sur les fonctionnalités disponibles, voir Référence des fonctionnalités Cockpit v2.

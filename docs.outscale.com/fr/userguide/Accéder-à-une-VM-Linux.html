Accéder à une VM Linux

Vous pouvez accéder à une machine virtuelle (VM) Linux en vous y connectant avec le protocole SSH.

Sur les ordinateurs Linux et macOS, le client OpenSSH est généralement installé par défaut. Sur Windows, vous devez l’installer.

Cette procédure est valable uniquement pour les VM issues d’OMI officielles, qui utilisent le système d’authentification par keypair par défaut. Dans d’autres cas, la méthode de connexion peut être différente. Pour en savoir plus, voir À propos des keypairs.

Depuis un ordinateur Linux ou macOS

Avant de commencer :

  1. Pour autoriser la connexion de votre ordinateur source vers la VM de destination, ajoutez la règle de security group suivante :

    • Port : 22 (SSH)

    • Protocole : TCP

    • CIDR : L’IP de votre ordinateur source

    Pour en savoir plus, voir Ajouter des règles à un security group.

  2. Dans un terminal, tapez la commande suivante pour restreindre à vous-même uniquement les autorisations de lecture/écriture de la keypair :

    $ chmod 600 KEYPAIR

    où KEYPAIR est le chemin vers le fichier .rsa contenant la clé privée de la keypair sur votre ordinateur.

Pour accéder à la VM avec OpenSSH, utilisez la commande ssh en suivant cette syntaxe :

$ ssh -i KEYPAIR outscale@VM_IP

Cette commande contient les attributs suivants que vous devez spécifier :

  • KEYPAIR : Le chemin vers le fichier .rsa contenant la clé privée de la keypair sur votre ordinateur.

  • VM_IP : L’IP publique de la VM.

    Si vous utilisez une connexion VPN ou DirectLink, vous pouvez spécifier l’IP privée de la VM.

Depuis un ordinateur Windows

Avant de commencer :

  1. Pour autoriser la connexion de votre ordinateur source vers la VM de destination, ajoutez la règle de security group suivante :

    • Port : 22 (SSH)

    • Protocole : TCP

    • CIDR : L’IP de votre ordinateur source

    Pour en savoir plus, voir Ajouter des règles à un security group.

  2. Dans un terminal, tapez les commandes suivantes pour restreindre à vous-même uniquement les autorisations de lecture/écriture de la keypair :

    $ Set filepath=KEYPAIR
$ icacls %filepath% /grant %username%:rw
$ icacls %filepath% /inheritance:d
$ icacls %filepath% /remove *S-1-5-11 *S-1-5-18 *S-1-5-32-544 *S-1-5-32-545

    où KEYPAIR est le chemin vers le fichier .rsa contenant la clé privée de la keypair sur votre ordinateur.

  3. Dans les Paramètres Windows, recherchez Ajouter une fonctionnalité facultative et installez Client OpenSSH.

Pour accéder à la VM avec OpenSSH, utilisez la commande ssh en suivant cette syntaxe :

$ ssh -i KEYPAIR outscale@VM_IP

Cette commande contient les attributs suivants que vous devez spécifier :

  • KEYPAIR : Le chemin vers le fichier .rsa contenant la clé privée de la keypair sur votre ordinateur.

  • VM_IP : L’IP publique de la VM.

    Si vous utilisez une connexion VPN ou DirectLink, vous pouvez spécifier l’IP privée de la VM.

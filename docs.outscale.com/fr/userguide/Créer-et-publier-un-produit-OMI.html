Créer et publier un produit OMI

En tant que vendeur partenaire sur OUTSCALE Marketplace, vous pouvez créer un produit image machine OUTSCALE (OMI) et le publier sur la Marketplace pour que les acheteurs puissent le consulter et y souscrire.

Lorsque vous créez un produit OMI, vous devez spécifier ses informations générales, configurer une ou plusieurs versions et une ou plusieurs offres, et joindre ou lier le contrat de licence. Vous pouvez également fournir un modèle permettant d’automatiser le déploiement du produit.

Par défaut, les produits OMI sont publiés sur cloudgouv-eu-west-1 et peuvent également être publiés sur eu-west-2.

Vous pouvez configurer une offre horaire et annuelle par type de VM, libre/open source, bring your own license (BYOL), ou une ou plusieurs offres abonnement pour votre produit OMI.

  • Une offre horaire et annuelle par type de VM combine le tarif du produit OMI et le tarif de la ou des machines virtuelles (VM) OUTSCALE choisies.

  • Dans le cas d’une offre libre/open source, seul le tarif de la ou des VM OUTSCALE choisies s’applique.

  • Dans le cas d’une offre BYOL, la facturation du produit OMI est gérée par le vendeur partenaire en dehors de la Marketplace, et le tarif de la ou des VM OUTSCALE choisies s’applique.

  • Une offre abonnement combine le tarif du produit OMI et le tarif de la ou des VM OUTSCALE choisies.

  • Vous devez d’abord créer votre vitrine avant de pouvoir créer un produit. Pour en savoir plus, voir Créer votre vitrine.

  • Vous devez compléter et enregistrer toutes les sections, dans l’ordre de votre choix, pour pouvoir publier le produit sur la Marketplace. Les sections à compléter sont marquées par un tag .

  • Lorsque vous créez un produit, vous devez fournir toutes les informations et tous les documents en français et en anglais dans les onglets donnés pour pouvoir publier votre produit sur la Marketplace.

Sélection du type de produit

Le type de produit et le mode de tarification ne pourront pas être modifiés ultérieurement.

  1. Dans le coin supérieur droit de l’interface, cliquez sur votre nom d’utilisateur ou votre adresse e-mail.
    Un menu déroulant apparaît.

  2. Cliquez sur Mes produits.
    La page Mes produits apparaît.

  3. Cliquez sur Créer un produit.
    La page Type de produit apparaît.

  4. Sélectionnez Image machine OUTSCALE (OMI).
    La liste de modes de tarification apparaît.

  5. Sélectionnez le mode de tarification, soit Par type de VM (à l’heure et à l’année), Bring your own license (BYOL), Libre/open source ou Abonnement.

  6. Cliquez sur Confirmer.
    La page de Création de votre produit apparaît, avec différentes sections à compléter.

Informations générales

  1. Dans la page Création de votre produit, cliquez sur Accéder dans la section Informations générales.
    Le formulaire Informations générales apparaît.

  2. Importer un logo pour votre produit.

    • Ce logo doit être un fichier de 2 Mo ou moins.

    • Nous recommandons d’importer un logo transparent et sans bordures pour un affichage correct.

  3. Saisissez un nom pour votre produit.

    Le nom du produit ne peut pas être modifié après la publication du produit.

  4. Sélectionnez une ou plusieurs catégories pour votre produit.

    • Vous pouvez sélectionner jusqu’à trois catégories pour votre produit.

    • Les catégories sélectionnées seront utilisées pour filtrer votre produit.

  5. Sélectionnez eu-west-2 si vous souhaitez publier le produit dans cette Région également.

    Les Régions ne peuvent pas être modifiées après la publication du produit.

  6. Dans la section Synthèse :

    1. Saisissez une description courte pour votre produit.

      Ce champ doit contenir entre un et 200 caractères.

    2. Saisissez une description longue pour votre produit.

    3. Saisissez les points clés de votre produit.

  7. (optionnel) Dans la section Médias :

    1. Importez les fichiers JPG souhaités pour illustrer votre produit.

      • Vous pouvez ajouter jusqu’à six fichiers JPG de 2 Mo ou moins.

      • Le format recommandé est 1440 pixels par 1024 pixels.

    2. Saisissez l’URL d’une vidéo YouTube.

  8. Dans la section Support :

    1. Saisissez les informations concernant le support de votre produit.

    2. Saisissez la politique de mise à jour de votre produit.

  9. Cliquez sur Enregistrer.
    Le formulaire est enregistré et un tag est affiché dans la section Informations générales.

  10. Si vous avez complété toutes les sections du produit et que vous êtes prêt à le publier :

    1. Dans la page Création de votre produit, cliquez sur Publier le produit pour publier votre produit.
      La boîte de dialogue Copie de l’OMI apparaît.

    2. Cliquez sur Poursuivre la publication.
      La demande de publication est envoyée à 3DS OUTSCALE et nécessite un délai de traitement. Le produit est au statut et ne pourra pas être modifié avant d’être au statut .

Versions

  1. Dans la page Création de votre produit, cliquez sur Accéder dans la section Versions.
    Le formulaire Versions apparaît.

  2. Dans la section Informations de l’OMI :

    1. Sélectionnez le nom et l’ID de l’OMI approprié.
      Le nom et l’ID de la copie de l’OMI sont automatiquement renseignés.

      Après avoir publié le produit, le nom et l’ID de l’OMI ne peuvent pas être modifiés, mais vous pouvez créer d’autres versions du produit avec d’autres noms et ID d’OMI depuis le tableau Versions. Pour en savoir plus, voir Gérer les versions d’un produit OMI.

      • Les OMI sélectionnables sont celles qui sont créées via le compte TINA correspondant et sur la Région correspondante.

      • Pour un fonctionnement optimal sur la Marketplace, ces OMI doivent rester privées. Pour en savoir plus, voir À propos des OMI.

    2. Saisissez la version du logiciel.

    3. Sélectionnez le système d’exploitation, soit Linux ou Windows.

    4. Donnez des précisions sur le système d’exploitation.

    5. Spécifiez la base logicielle en ajoutant des éléments.

    6. Saisissez une note de release.

  3. Dans la section Informations de configuration, saisissez les instructions d’utilisation de votre produit.

  4. Cliquez sur Enregistrer.
    Le formulaire est enregistré et un tag est affiché dans la section Versions.

  5. Si vous avez complété toutes les sections du produit et que vous êtes prêt à le publier :

    1. Dans la page Création de votre produit, cliquez sur Publier le produit pour publier votre produit.
      La boîte de dialogue Copie de l’OMI apparaît.

    2. Cliquez sur Poursuivre la publication.
      La demande de publication est envoyée à 3DS OUTSCALE et nécessite un délai de traitement. Le produit est au statut et ne pourra pas être modifié avant d’être au statut .

Légal

  1. Dans la page Création de votre produit, cliquez sur Accéder dans la section Légal. Le formulaire Légal apparaît.

  2. Dans le cas d’un produit avec un mode de tarification horaire et annuel par type de VM ou abonnement :

    1. Cliquez sur Importer pour importer le contrat de licence de votre produit.

      Le contrat de licence doit être un fichier PDF de 6 Mo ou moins.

    2. Cliquez sur Enregistrer.
      Les contrats de licence anglais et français sont joints à votre produit et un tag est affiché dans la section Légal.

    Dans le cas d’un produit avec un mode de tarification BYOL ou libre/open source :

    1. Saisissez l’URL du contrat de licence de votre produit.

    2. Cliquez sur Enregistrer.
      Un tag est affiché dans la section Légal.

  3. Si vous avez complété toutes les sections du produit et que vous êtes prêt à le publier :

    1. Dans la page Création de votre produit, cliquez sur Publier le produit pour publier votre produit.
      La boîte de dialogue Copie de l’OMI apparaît.

    2. Cliquez sur Poursuivre la publication.
      La demande de publication est envoyée à 3DS OUTSCALE et nécessite un délai de traitement. Le produit est au statut et ne pourra pas être modifié avant d’être au statut .

Offre

Le tarif du logiciel ne peut pas être modifié après la publication du produit, sauf le tarif annuel d’une offre par type de VM.

  1. Dans la page Création de votre produit, cliquez sur Accéder dans la section Offres.
    Le formulaire Offres apparaît.

  2. Si vous avez sélectionné un mode de tarification Bring your own license (BYOL) ou Libre/open source :

    1. Dans la section Paramètres de votre offre :

      1. Saisissez un nom pour votre offre.

      2. Saisissez une description pour votre offre.

        • En mode libre/open source et BYOL, aucun tarif produit n’est applicable.

    2. Dans la section VM recommandées, ajoutez au moins une VM recommandée pour votre produit :

      1. Cliquez sur Ajouter une VM recommandée.
        La fenêtre Ajouter une VM recommandée apparaît.

      2. Spécifiez les paramètres de la VM.

      3. Cliquez sur Enregistrer.
        Le type de VM que vous venez de renseigner est ajouté dans le tableau.

        • Vous pouvez modifier les VM recommandées en cliquant sur Modifier pour la VM souhaitée.

        • Vous pouvez supprimer les VM recommandées en cliquant sur Supprimer pour la VM souhaitée.

        Pour en savoir plus sur les types de VM, voir Types de VM.
    Si vous avez sélectionné un mode de tarification Par type de VM (à l’heure et à l’année) :

    1. Dans la section Paramètres de votre offre :

      1. Saisissez un nom pour votre offre.

      2. Saisissez une description pour votre offre.

    2. Dans la section Paramètres de votre tarif :

      1. Cliquez sur Ajouter un type de VM.
        La fenêtre Ajouter un type de VM apparaît.

      2. Spécifiez les paramètres de la VM.

      3. Cliquez sur Enregistrer.
        Une ligne est ajoutée dans le tableau avec les informations que vous venez de renseigner.

      4. Cliquez sur le champ Licence/heure pour spécifier un tarif hors taxes horaire pour votre licence, en fonction du type de VM paramétré.

      5. Cliquez sur le champ Licence/an pour spécifier un tarif hors taxes annuel pour votre licence, en fonction du type de VM paramétré.

        • Vous pouvez ajouter autant de types de VM que nécessaire.

        • Vous ne pouvez pas ajouter deux fois un même type de VM.

        • Pour en savoir plus sur les types de VM, voir Types de VM.

        • Vous pouvez modifier les types de VM en cliquant sur Modifier pour le type de VM souhaité.

        • Vous pouvez supprimer les types de VM en cliquant sur Supprimer pour le type de VM souhaité.

        Vous ne pouvez pas modifier ou supprimer un type de VM une fois que l’offre est publiée.

      6. Spécifiez un tarif par défaut hors taxes pour la licence.

        Le tarif par défaut est appliqué lorsqu’un acheteur utilise votre produit sur une VM qui n’est pas incluse dans les types de VM de votre offre.
    Si vous avez sélectionné un mode de tarification Abonnement :

    Vous pouvez créer plusieurs offres d’abonnement.

    1. Dans la page Offres, cliquez sur Créer une offre.

      1. Dans la section Paramètres de votre offre :

        1. Saisissez un nom pour votre offre.

        2. Saisissez une description pour votre offre.

      2. Dans la section Tarif de votre offre :

        1. Sélectionnez une durée d’engagement, soit 1 mois, 1 an, 2 ans, ou 3 ans.

        2. Sélectionnez la fréquence de facturation, soit À la souscription, Au mois, ou À l’année.

        3. Définissez un taux hors taxes pour votre offre.

      3. Dans la section VM recommandées, ajoutez au moins une VM recommandée pour votre produit :

        1. Cliquez sur Ajouter une VM recommandée.
          La fenêtre Ajouter une VM recommandée apparaît.

        2. Spécifiez les paramètres de la VM.

        3. Cliquez sur Enregistrer.
          Le type de VM que vous venez de renseigner est ajouté dans le tableau.

          • Vous pouvez modifier les VM recommandées en cliquant sur Modifier pour la VM souhaitée.

          • Vous pouvez supprimer les VM recommandées en cliquant sur Supprimer pour la VM souhaitée.

          Pour en savoir plus sur les types de VM, voir Types de VM.

  3. Cliquez sur Enregistrer.
    Le formulaire est enregistré et un tag est affiché dans la section Offre.

  4. Si vous avez complété toutes les sections du produit et que vous êtes prêt à le publier :

    1. Dans la page Création de votre produit, cliquez sur Publier le produit pour publier votre produit.
      La boîte de dialogue Copie de l’OMI apparaît.

    2. Cliquez sur Poursuivre la publication.
      La demande de publication est envoyée à 3DS OUTSCALE et nécessite un délai de traitement. Le produit est au statut et ne pourra pas être modifié avant d’être au statut .

Modèle de déploiement

La section Modèle de déploiement est optionnelle.

  1. Dans la page Création de votre produit, cliquez sur Accéder dans la section Modèle de déploiement.
    Le formulaire Modèle de déploiement apparaît.

  2. Saisissez l’URL de dépôt Git vers le modèle.

  3. Saisissez une description du modèle.

  4. Saisissez les instructions d’utilisation du modèle.

    Vous pouvez ajouter jusqu’à six fichiers JPG de 2 Mo ou moins.

  5. (optionnel) Importez un schéma d’architecture.

    Le schéma d’architecture doit être un fichier PNG de 2 Mo ou moins.

  6. Cliquez sur Enregistrer.
    Le formulaire est enregistré et un tag est affiché dans la section Modèle de déploiement.

  7. Si vous avez complété toutes les sections du produit et que vous êtes prêt à le publier :

    1. Dans la page Création de votre produit, cliquez sur Publier le produit pour publier votre produit.
      La boîte de dialogue Copie de l’OMI apparaît.

    2. Cliquez sur Poursuivre la publication.
      La demande de publication est envoyée à 3DS OUTSCALE et nécessite un délai de traitement. Le produit est au statut et ne pourra pas être modifié avant d’être au statut .

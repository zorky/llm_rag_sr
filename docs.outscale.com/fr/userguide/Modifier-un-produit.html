Modifier un produit

En tant que vendeur partenaire sur OUTSCALE Marketplace, vous pouvez modifier des produits publiés sur la Marketplace ou à l’état brouillon.

  • Les modifications apportées à vos produits ne s’appliquent pas aux utilisateurs qui y ont déjà souscrit, mais uniquement aux nouvelles souscriptions. Cependant, lorsque vous modifiez les contrats de licence de vos produits OMI, vous pouvez choisir d’appliquer les modifications aux souscripteurs existants en plus des nouvelles souscriptions. Dans ce cas, les souscripteurs existants sont informés par e-mail, et les contrats de licence sont mis à jour dans leur historique produit. Si vous choisissez de ne pas appliquer les modifications aux souscripteurs existants, les modifications ne s’appliqueront qu’aux nouvelles souscriptions.

  • Toute modification apportée à un produit doit être faite en anglais et en français pour que les changements soient publiés sur la Marketplace.

Pour modifier un produit :

  1. Dans le coin supérieur droit de l’interface, cliquez sur votre nom d’utilisateur ou votre adresse e-mail.
    Un menu déroulant apparaît.

  2. Cliquez sur Mes produits.
    La page Mes produits apparaît.

  3. Cliquez sur Modifier sur la ligne produit souhaitée.
    La page produit correspondante apparaît.

  4. Cliquez sur Accéder dans la section souhaitée pour modifier les informations.
    Le formulaire correspondant apparaît.

    Les éléments suivants ne peuvent pas être modifiés dans un produit OMI publié :

    • Le type de produit et le mode de tarification sélectionnés à la création.

    • Le nom du produit dans la section Informations générales.

    • L’ID ou le nom de l’OMI dans la section Versions.

    • Le taux hors taxes dans la section Offre.

    • Les Régions dans lesquelles le produit est publié.

    Vous pouvez publier de nouvelles versions de votre produit OMI avec de nouveaux noms ou ID d’OMI. Pour en savoir plus, voir Gérer les versions d’un produit OMI.

  5. Modifiez les informations souhaitées et cliquez sur Publier les modifications.
    Les modifications sont publiées sur la Marketplace et la colonne Dernière mise à jour du produit est mise à jour dans la page Mes produits.

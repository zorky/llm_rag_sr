Se connecter à Cockpit v2

Vous pouvez vous connecter à Cockpit v2, soit en tant que compte racine, soit en tant qu’utilisateur Elastic Identity Management (EIM), via différentes méthodes et avec différents degrés de sécurité.

Vous pouvez vous connecter à Cockpit v2 avec l’authentification à facteur simple. Cela signifie que vous pouvez vous authentifier en seulement une étape, avec deux méthodes possibles : votre identifiant et votre mot de passe, ou vos access keys.

Si vous ne parvenez pas à vous authentifier correctement quatre fois de suite, les tentatives d’authentification suivantes seront verrouillées pendant 1 minute. Ce temps de verrouillage augmente d'1 minute toutes les quatre tentatives infructueuses consécutives, et ce jusqu’à 10 minutes.

Vous pouvez également renforcer la sécurité de votre compte en utilisant l’authentification multifacteur (MFA). Cette fonctionnalité ajoute une étape d’authentification supplémentaire à celle de l’authentification à facteur simple en nécessitant soit une clé de sécurité (via USB ou intégrée à un smartphone ou ordinateur), soit un mot de passe à usage unique (OTP).

Pour en savoir plus, voir À propos de l’authentification.

Authentification à facteur simple

En tant que compte racine

Avec un identifiant et un mot de passe

  1. Accédez à Cockpit v2.

  2. Sélectionnez la Région où se trouve votre compte OUTSCALE.

  3. Dans le champ Identifiant, entrez votre adresse e-mail.

  4. Cliquez sur Se connecter.
    Le champ Mot de passe apparaît.

  5. Dans le champ Mot de passe, entrez votre mot de passe.

  6. Cliquez sur Se connecter.

Avec vos access keys

  1. Accédez à Cockpit v2.

  2. Sélectionnez la Région où se trouve votre compte OUTSCALE.

  3. Cliquez sur Essayer une autre méthode.

  4. Cliquez sur Access key / Secret key.

  5. Dans le champ Access key, entrez l’ID de votre access key.

  6. Dans le champ Secret key, entrez la secret key correspondante.

  7. Cliquez sur Se connecter.

En tant qu’utilisateur EIM

Avec un identifiant et un mot de passe

Avant de commencer : En tant qu’utilisateur EIM, vous n’avez initialement que l’option de vous connecter à Cockpit v2 avec l’AK/SK créée pour vous par le compte racine. Une fois connecté à Cockpit v2, vous pouvez créer votre propre mot de passe. Pour en savoir plus, voir Créer un mot de passe en tant qu’utilisateur EIM.

  1. Accédez à Cockpit v2.

  2. Sélectionnez la Région où se trouve votre compte OUTSCALE.

  3. Dans le champ Nom d’utilisateur, entrez votre identifiant.

    Votre identifiant suit le format suivant : ID du compte racine/votre nom d’utilisateur (par exemple : 012345678912/username).

  4. Cliquez sur Se connecter.
    Le champ Mot de passe apparaît.

  5. Dans le champ Mot de passe, entrez votre mot de passe.

  6. Cliquez sur Se connecter.

Avec vos access keys

  1. Accédez à Cockpit v2.

  2. Sélectionnez la Région où se trouve votre compte OUTSCALE.

  3. Cliquez sur Essayer une autre méthode.

  4. Cliquez sur Access key / Secret key.

  5. Dans le champ Access key, entrez l’ID de votre access key.

  6. Dans le champ Secret key, entrez la secret key correspondante.

  7. Cliquez sur Se connecter.

Authentification multifacteur

L’authentification multifacteur peut être utilisée de la même manière à la fois par le compte racine et par les utilisateurs EIM.
Pour utiliser l’authentification multifacteur, vous devez vous authentifier avec une des méthodes de la section précédente puis avec une des méthodes ci-dessous.

Se connecter et se déconnecter avec WebAuthn

  • WebAuthn est un standard web se basant sur la cryptographie à clé publique, qui permet de s’authentifier à des applications web à l’aide d’une clé de sécurité (via USB, comme une YubiKey, ou intégrée à un smartphone ou tablette) ou à l’aide de l’authentification biométrique de votre ordinateur ou smartphone.

  • Un OTP est une courte séquence de chiffres autogénérés fournie par une application. Cette séquence n’est disponible que pendant une courte période, rendant ainsi plus difficile toute utilisation malveillante.

Avant de commencer : Configurez la MFA pour votre compte OUTSCALE. Pour en savoir plus, voir Mettre en place une MFA pour votre compte avec WebAuthn ou un OTP.

  1. Accédez à Cockpit v2.

  2. Sélectionnez la Région où se trouve votre compte OUTSCALE.

  3. Entrez vos identifiants et cliquez sur Se connecter.
    La page Se connecter avec un second facteur apparaît.

  4. Si vous n’avez configuré qu’un seul périphérique, cliquez sur Se connecter. Si vous avez configuré plusieurs périphériques, sélectionnez celui approprié et cliquez sur Se connecter.
    Une boîte de dialogue de votre navigateur apparaît.

  5. Suivez les instructions de votre navigateur.

  6. Pour vous déconnecter, cliquez sur l’icône dans le coin supérieur droit, puis Se déconnecter.

Se connecter et se déconnecter avec un mot de passe à usage unique (OTP)

  • WebAuthn est un standard web se basant sur la cryptographie à clé publique, qui permet de s’authentifier à des applications web à l’aide d’une clé de sécurité (via USB, comme une YubiKey, ou intégrée à un smartphone ou tablette) ou à l’aide de l’authentification biométrique de votre ordinateur ou smartphone.

  • Un OTP est une courte séquence de chiffres autogénérés fournie par une application. Cette séquence n’est disponible que pendant une courte période, rendant ainsi plus difficile toute utilisation malveillante.

Avant de commencer : Configurez la MFA pour votre compte OUTSCALE. Pour en savoir plus, voir Mettre en place une MFA pour votre compte avec WebAuthn ou un OTP.

  1. Accédez à Cockpit v2.

  2. Sélectionnez la Région où se trouve votre compte OUTSCALE.

  3. Entrez vos identifiants et cliquez sur Se connecter.
    La page Se connecter avec un second facteur apparaît.

  4. Sur votre téléphone, ouvrez l’application d’authentification choisie.

  5. Dans le champ Mot de passe à usage unique, entrez l’OTP fourni par l’application.
    Vous êtes connecté à Cockpit v2.

  6. Pour vous déconnecter, cliquez sur l’icône dans le coin supérieur droit, puis Se déconnecter.

Rechercher dans la documentation publique depuis votre barre d’URL

Cette page explique comment ajouter la documentation publique OUTSCALE aux moteurs de recherche de votre navigateur.

Cela vous permet de rechercher rapidement dans le site de documentation à l’aide d’un raccourci/mot-clé. Par exemple, vous pouvez taper osc vm types dans votre barre d’URL pour lancer directement une recherche pour "vm types" dans la documentation, même si vous n’êtes pas déjà sur le site.

Firefox

  1. Faites un clic droit sur la barre de recherche située à l’intérieur de n’importe quelle page de la documentation publique OUTSCALE, puis cliquez sur Ajouter un mot-clé pour cette recherche…​.
    Une fenêtre apparaît.

  2. Remplissez la fenêtre comme suit :

    • Nom : Le nom que vous voulez pour le moteur de recherche (par exemple, OUTSCALE Docs (FR)).

    • Emplacement : Le répertoire où vous voulez ranger le moteur de recherche dans vos marque-pages.

    • Mot-clé : Le mot-clé que vous voulez utiliser pour appeler le moteur de recherche (par exemple, osc).

  3. Cliquez sur Enregistrer pour confirmer.
    Vous pouvez maintenant faire des recherches dans la documentation directement depuis votre barre d’URL, en tapant le mot-clé osc suivi de votre requête.

  • Pour en savoir plus, voir la documentation officielle de Firefox.

  • Vous pouvez aussi suivre une autre méthode aussi décrite dans la documentation officielle de Firefox. Dans ce cas, le moteur de recherche sera stocké dans les réglages de votre navigateur plutôt que dans vos marque-pages.

Google Chrome

  1. Dans le menu Paramètres de votre navigateur, cliquez sur Moteur de recherche > Gérer les moteurs de recherche et la recherche sur les sites.

    Vous pouvez aussi taper chrome://settings/searchEngines dans votre barre d’URL.

  2. Cliquez sur Ajouter.
    Une fenêtre apparaît.

  3. Remplissez la fenêtre comme suit :

    • Moteur de recherche : Le nom que vous voulez pour le moteur de recherche (par exemple, OUTSCALE Docs (FR)).

    • Raccourci : Le mot-clé que vous voulez utiliser pour appeler le moteur de recherche (par exemple, osc).

    • URL avec %s à la place de la requête : Tapez https://docs.outscale.com/fr/userguide/Recherche.html?q=%s.

  4. Cliquez sur Ajouter pour confirmer.
    Vous pouvez maintenant faire des recherches dans la documentation directement depuis votre barre d’URL, en tapant le mot-clé osc suivi de votre requête.

Pour en savoir plus, voir la documentation officielle de Google Chrome.

Microsoft Edge

  1. Dans le menu Paramètres de votre navigateur, cliquez sur Confidentialité, recherche et services > Barre d’adresse et recherche > Gérer les moteurs de recherche.

    Vous pouvez aussi taper edge://settings/searchEngines dans votre barre d’URL.

  2. Cliquez sur Ajouter.
    Une fenêtre apparaît.

  3. Remplissez la fenêtre comme suit :

    • Moteur de recherche : Le nom que vous voulez pour le moteur de recherche (par exemple, OUTSCALE Docs (FR)).

    • Raccourci : Le mot-clé que vous voulez utiliser pour appeler le moteur de recherche (par exemple, osc).

    • URL avec %s à la place de la requête : Tapez https://docs.outscale.com/fr/userguide/Recherche.html?q=%s.

  4. Cliquez sur Ajouter pour confirmer.
    Vous pouvez maintenant faire des recherches dans la documentation directement depuis votre barre d’URL, en tapant le mot-clé osc suivi de votre requête.

Pour en savoir plus, voir la documentation officielle de Microsoft Edge.

Safari

Avec Safari, vous devez utiliser une extension de navigateur pour activer la recherche par mot-clé. Par exemple, voir http://safarikeywordsearch.aurlien.net.

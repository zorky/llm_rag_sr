Tarification des produits OMI

Lorsque vous souscrivez à un produit OMI sur OUTSCALE Marketplace, vous payez le tarif du logiciel selon le mode de tarification sélectionné par le vendeur partenaire, ainsi que le tarif d’une ou de plusieurs machines virtuelles (VM) OUTSCALE.

Vous pouvez souscrire à des produits OMI sur la base d’une offre libre/open source, bring your own license (BYOL), ou payante.

Vous pouvez estimer le coût d’un produit image machine OUTSCALE (OMI) utilisé sur une ou plusieurs machines virtuelles (VM) avant d’y souscrire. Pour en savoir plus, voir Estimer le coût d’un produit.

Les coûts indiqués pour la réservation de VM n’incluent pas le stockage. Pour en savoir plus, voir Block Storage Unit (BSU) et la grille tarifaire de 3DS OUTSCALE.

Offres libres/open source

Lorsque vous souscrivez à une offre libre/open source, aucuns frais ne s’appliquent au produit, et vous ne payez que le tarif d’une ou de plusieurs VM OUTSCALE.

Offres bring your own license (BYOL)

Dans le cadre d’une offre BYOL, vous souscrivez au logiciel du vendeur partenaire directement via son site web, sur la base de sa tarification, et vous utilisez le produit sur une ou plusieurs VM OUTSCALE.

Vous êtes facturé d’une part par le vendeur partenaire pour son logiciel, et d’une autre par OUTSCALE pour l’utilisation de VM.

Offres payantes

Offres d’abonnements

Lorsque vous souscrivez à une offre abonnement, vous payez le tarif du logiciel et le tarif d’une ou de plusieurs VM OUTSCALE.

Vous êtes soumis à une période d’engagement et une fréquence de facturation définies par le vendeur partenaire pour son logiciel.

  • Le tarif du logiciel reste fixe, quel que soit le type de VM utilisé.

  • Si vous réservez une ou plusieurs VM, la durée d’engagement et la fréquence de facturation de la réservation de l’abonnement sont automatiquement alignées.

Offres de services LLM Mistral

Lorsque vous souscrivez à un service LLM Mistral, vous payez le coût du modèle Mistral souhaité ainsi que le coût des ressources Cloud OUTSCALE correspondantes. Le service est facturé à partir de la date de déploiement sur une base mensuelle. Tout mois entamé est facturé entièrement.

Pour connaître le coût des différents modèles Mistral, voir les tarifs publics OUTSCALE.

Offres horaires et annuelles

Offres horaires et annuelles par type de VM

Lorsque vous souscrivez à une offre horaire et annuelle par type de VM, votre durée d’engagement pour le logiciel détermine votre mode de facturation.

  • Si vous choisissez de consommer à la demande, vous payez le tarif horaire de l’offre.

  • Si vous choisissez de consommer avec une durée d’engagement de un an, vous payez le tarif annuel de l’offre.

Le tarif du logiciel évolue en fonction de la VM sélectionnée.

Le coût de l’offre est valable pour le type de VM sélectionné lors de la souscription. Si votre utilisation diffère des conditions de votre réservation de VM, vous serez facturé en supplément, à l’heure, selon les tarifs mentionnés sur la page produit.

Offres de licences Microsoft Windows

Lorsque vous souscrivez à une offre de licence Miscrosoft Windows, le type de VM sélectionné détermine le nombre de licences auxquelles vous souscrivez.

Si vous ne réservez pas de VM, vous serez facturé le nombre de licences Microsoft Windows correspondant au type de VM consommé.

Pour en savoir plus sur la facturation, voir le site officiel de 3DS OUTSCALE et le centre d’aide OUTSCALE.

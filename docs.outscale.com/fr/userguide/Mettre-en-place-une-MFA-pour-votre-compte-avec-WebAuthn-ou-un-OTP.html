Mettre en place une MFA pour votre compte avec WebAuthn ou un OTP

Avec Cockpit v2, vous pouvez mettre en place une authentification multifacteur (MFA) pour votre compte OUTSCALE afin d’en renforcer la sécurité. La MFA combine une méthode d’authentification primaire soit avec une clé de sécurité ou des données biométriques (WebAuthn), soit avec un mot de passe à usage unique (one-time password, ou OTP).

  • WebAuthn est un standard web se basant sur la cryptographie à clé publique, qui permet de s’authentifier à des applications web à l’aide d’une clé de sécurité (via USB, comme une YubiKey, ou intégrée à un smartphone ou tablette) ou à l’aide de l’authentification biométrique de votre ordinateur ou smartphone.

  • Un OTP est une courte séquence de chiffres autogénérés fournie par une application. Cette séquence n’est disponible que pendant une courte période, rendant ainsi plus difficile toute utilisation malveillante.

Pour en savoir plus sur l’authentification, voir À propos de l’authentification.

Nous vous recommandons de configurer des codes de récupération qui vous permettent de vous authentifier quand vous êtes dans l’incapacité d’utiliser la MFA. Pour en savoir plus, voir S’authentifier avec des codes de récupération.

Si vous perdez l’unique appareil que vous utilisez, et que vous n’avez pas configuré ou avez perdu vos codes de récupération, veuillez contacter notre équipe Support. Pour en savoir plus, voir Support technique.

WebAuthn

Cette méthode d’authentification ne fonctionne pas sur Firefox avec macOS.

Avant de commencer :

Assurez-vous d’avoir un des appareils compatibles suivants :

  • Une clé de sécurité USB (YubiKey)

  • Un smartphone ou une tablette avec clé de sécurité intégrée

  • Un appareil compatible avec les données biométriques

Vous pouvez enregistrer autant d’appareils que nécessaire.

  1. Dans le coin supérieur droit, cliquez sur l’icône .
    Un menu déroulant apparaît.

  2. Cliquez sur  Paramètres d’authentification.
    La page Paramètres d’authentification apparaît.

  3. Dans la section Authentification multifacteur, cliquez sur Configurer une clé de sécurité.
    La page Configurer une clé de sécurité apparaît.

  4. Cliquez sur Configurer.
    Une boîte de dialogue de votre navigateur apparaît.

  5. Suivez les instructions de votre navigateur pour configurer votre périphérique.
    Vous avez le choix entre enregistrer une clé de sécurité USB, un smartphone ou une tablette avec clé de sécurité intégrée (via Bluetooth), ou un ordinateur avec authentification biométrique.

Mot de passe à usage unique

Avant de commencer :

Installez une application d’authentification sur votre smartphone ou sur un autre appareil, de préférence un appareil différent de celui utilisé pour naviguer sur Cockpit v2.

Vous pouvez enregistrer autant d’appareils que nécessaire.

  1. Dans le coin supérieur droit, cliquez sur l’icône .
    Un menu déroulant apparaît.

  2. Cliquez sur  Paramètres d’authentification.
    La page Paramètres d’authentification apparaît.

  3. Dans la section Authentification multifacteur, cliquez sur Configurer une application d’authentification.
    La page Configurer une application d’authentification apparaît.

  4. Ouvrez l’application d’authentification sur votre appareil et scannez le QR code affiché.

    Si vous ne parvenez pas à scanner le QR code, cliquez sur Impossible de scanner ? pour afficher une clé à saisir manuellement dans votre application d’authentification.

  5. Dans le champ Mot de passe à usage unique, entrez les six chiffres fournis par l’application.

  6. (optionnel) Dans le champ Nom du périphérique, entrez un nom pour l’appareil correspondant.

  7. Cliquez sur Configurer pour valider.

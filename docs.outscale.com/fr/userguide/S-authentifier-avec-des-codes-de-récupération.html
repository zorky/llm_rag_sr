S’authentifier avec des codes de récupération

Vous pouvez utiliser un code de récupération comme deuxième facteur d’authentification pour vous connecter à Cockpit v2, dans le cas où votre méthode d’authentification multifacteur (MFA) privilégiée n’est pas disponible.

Informations générales

Les codes de récupération se présentent sous la forme d’une liste de 12. Chaque code ne peut être utilisé qu’une seule fois. Une nouvelle liste peut être générée, que vous ayez utilisé les 12 codes ou non. Pour en savoir plus, voir Configurer des codes de récupération.

Assurez-vous d’imprimer, télécharger, ou copier les codes de récupération car vous ne pourrez pas les récupérer plus tard.

Les codes de récupération ne peuvent être utilisé qu’à la place d’une méthode de MFA.

Configurer des codes de récupération

  1. Connectez-vous à Cockpit v2.

  2. Dans le coin supérieur droit, cliquez sur l’icône .
    Un menu déroulant apparaît.

  3. Cliquez sur Paramètres d’authentification.
    La page Paramètres d’authentification apparaît.

  4. Dans la section Authentification multifacteur, cliquez sur Configurer des codes de récupération.
    Une liste de codes de récupération apparaît.

    La liste est seulement disponible pendant la configuration. Assurez-vous d’imprimer, télécharger, ou copier les codes de récupération car vous ne pourrez pas les récupérer plus tard.

  5. Cliquez sur Configurer.
    Les codes de récupération sont configurés pour votre compte.

Utiliser les codes de récupération

Si vous tentez de vous authentifier avec un code de récupération non valide quatre fois de suite, les tentatives d’authentification suivantes seront verrouillées pendant 1 minute. Ce temps de verrouillage augmente d'1 minute toutes les tentatives infructueuses consécutives, et ce jusqu’à 10 minutes.

  1. Accédez à Cockpit v2.

  2. Sélectionnez la Région où se trouve votre compte OUTSCALE.
    La page de connexion apparaît.

  3. Entrez vos identifiants et cliquez sur Se connecter.
    La page Se connecter avec un second facteur apparaît.

  4. Cliquez sur Essayer une autre méthode.

  5. Cliquez sur Code de récupération.

  6. Dans le champ Code de récupération, tapez le code de récupération correspondant au numéro donné et cliquez sur Se connecter.

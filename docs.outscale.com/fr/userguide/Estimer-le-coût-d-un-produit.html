Estimer le coût d’un produit

En tant que visiteur ou acheteur sur OUTSCALE Marketplace, vous pouvez estimer le coût d’un produit image machine OUTSCALE (OMI) utilisé sur une ou plusieurs machines virtuelles (VM) avant d’y souscrire.

Vous pouvez choisir d’estimer le coût du produit utilisé sur des VM OUTSCALE recommandées par le vendeur partenaire, ou sur des VM OUTSCALE personnalisables. L’estimation peut porter sur la Région cloudgouv-eu-west-1, et éventuellement sur eu-west-2 si le produit a été publié dans cette seconde Région.

  • Cette estimation ne constitue pas un devis ou une facture. Elle est valide pour une durée limitée et le coût indiqué est susceptible d’évoluer.

  • Pour obtenir une estimation valide dans le cadre du marché Nuage public, voir Souscrire à une offre Nuage public (UGAP).

  • Le coût de VM indiqué n’inclut pas le stockage. Pour en savoir plus, voir Block Storage Unit (BSU) et la grille tarifaire de 3DS OUTSCALE.

  • Lorsque vous estimez le coût d’un produit sur la base d’une durée d’engagement, vous estimez le coût du produit utilisé sur une ou plusieurs VM réservées. Pour en savoir plus sur les VM réservées et les remises associées, voir À propos des VM > VM réservées et notre site officiel.

  • Pour en savoir plus sur la tarification des produits, voir Tarification des produits OMI.

Pour estimer le coût d’un produit :

  1. Dans l’interface Marketplace, cliquez sur l’onglet Catalogue.
    Le catalogue de produits apparaît.

  2. Utilisez les filtres à gauche pour trouver le produit souhaité.

  3. Cliquez sur la vignette du produit souhaité.
    La page du produit apparaît.

  4. Si vous souhaitez souscrire à une offre abonnement :

    1. Dans la section Logiciel, sélectionnez l’offre souhaitée s’il en existe plusieurs.

      • La durée d’engagement et la fréquence de facturation sont automatiquement définies sur la base de l’offre sélectionnée.

      • Vous pouvez directement passer à l’étape 7 de la procédure.
    Si vous souhaitez souscrire à une offre horaire et annuelle par type de VM :

    1. Dans la section Logiciel, sélectionnez la durée d’engagement, soit Sans engagement soit 1 an.

    2. Si vous choisissez de vous engager sur un an, sélectionnez le nombre de licences auquel vous souhaitez souscrire.
      Vous pouvez ensuite directement passer à l’étape 7 de la procédure.

    Si vous souhaitez souscrire à un autre type d’offre du catalogue, aucun paramétrage n’est nécessaire à cette étape. Vous pouvez directement passer à l’étape suivante.

  5. Dans la section Configuration des VM, sélectionnez votre durée d’engagement, soit Sans engagement, 1 mois, 1 an, 2 ans, ou 3 ans.

    La durée d’engagement sélectionnée détermine les fréquences de facturation disponibles à l’étape suivante.

  6. Sélectionnez votre fréquence de facturation, soit À la souscription, Au mois, ou À l’année.

  7. Sélectionnez la Région de déploiement, soit cloudgouv-eu-west-1 ou eu-west-2.

  8. Sélectionnez un type de VM, soit VM recommandée ou VM personnalisable.

    Si vous avez sélectionné VM recommandée :

    1. Sélectionnez une VM recommandée par le vendeur partenaire.

      Les types de VM listés sont recommandés par le vendeur partenaire pour un déploiement viable du produit. Pour en savoir plus sur une VM recommandée, vous pouvez survoler le type de VM en bleu.
    Si vous avez sélectionné VM personnalisable :

    1. Configurez le type de VM souhaité.
      Le type de VM et les prix sont mis à jour.

      Pour en savoir plus sur les types de VM, voir Types de VM.

  9. Sélectionnez le nombre de VM.

    Si vous souscrivez à une offre annuelle par type de VM, le nombre de licences sélectionné à l’étape 4 détermine le nombre de VM. Vous n’avez donc aucune sélection à effectuer pour cette étape.

    Le coût du logiciel et de VM est mis à jour dans la section Estimation à droite.

Configurer Cyberduck pour OOS

Vous pouvez installer Cyberduck, une interface web open source, pour utiliser OUTSCALE Object Storage (OOS).

3DS OUTSCALE fournit des fichiers XML que vous pouvez utiliser pour configurer automatiquement Cyberduck.

Pour en savoir plus sur l’utilisation de Cyberduck, voir Official Cyberduck documentation for S3 (disponible en anglais uniquement).

Configurer Cyberduck pour le stockage objet sur macOS

Installer Cyberduck sur votre machine

Avant de commencer : Téléchargez et installez Cyberduck.

  1. Accédez au répertoire Bookmarks de Cyberduck :

    $ cd "/${HOME}/Library/Group Containers/G69SCX94XU.duck/Library/Application Support/duck/Bookmarks/"

  2. Cliquez ici pour télécharger le fichier de configuration nécessaire à l’interaction avec OOS.

Le fichier devrait être automatiquement ajouté à la section signet de Cyberduck. Si ce n’est pas le cas, copiez le fichier de configuration nouvellement téléchargé dans votre dossier Bookmarks :

$ cp "/${HOME}/Downloads/Outscale_oos_signed_macOS.duck" "/${HOME}/Library/Group Containers/G69SCX94XU.duck/Library/Application Support/duck/Bookmarks/"

Configurer OOS sur Cyberduck

  1. Ouvrez l’application Cyberduck.

  2. Dans la fenêtre principale, double-cliquez sur la connexion OUTSCALE Object Storage. Une boîte de dialogue apparaît.

  3. Spécifiez vos identifiants :

    1. Dans le champ Access Key ID, tapez l’ID de votre access key.

    2. Dans le champ Secret Access Key, tapez la secret key associée à l’ID de votre access key.

      Pour obtenir l’ID de votre access key et votre secret key pour autoriser la connexion aux API, voir Obtenir des informations sur vos access keys.

  4. Cliquez sur Connect pour valider.
    Vous êtes connecté à vos ressources OOS, et vos buckets et fichiers apparaissent dans l’interface.

    • Pour configurer manuellement une connexion OOS, vous pouvez cliquer sur Open Connection . Dans ce cas, ne spécifiez pas "https" lorsque vous tapez le endpoint qui correspond à la Région que vous voulez utiliser.

    • Vous pouvez désactiver les notifications "Bonjour" de Cyberduck en exécutant la commande suivante dans votre terminal :

      $ defaults write ch.sudo.cyberduck rendezvous.enable false

    • Vous pouvez réactiver les notifications "Bonjour" en exécutant la même commande avec true au lieu de false.

Configurer Cyberduck pour le stockage objet sur Windows

Installer Cyberduck sur votre machine

Avant de commencer : Téléchargez et installez Cyberduck.

  1. Accédez au répertoire Bookmarks de Cyberduck :

    $ C:\Users\[YourUsername]r\AppData\Roaming\Cyberduck\Bookmarks

  2. Cliquez ici pour télécharger le fichier de configuration nécessaire à l’interaction avec OOS.

  3. Copiez ce fichier dans le dossier relatif aux signets :

    $ copy "C:\Users\[YourUsername]\Downloads\Outscale_oos_signed_Windows.duck" "C:\Users\[YourUsername]\AppData\Roaming\Cyberduck\Bookmarks"

Configurer OOS sur Cyberduck

  1. Ouvrez l’application Cyberduck.

  2. Dans la fenêtre principale, double-cliquez sur la connexion OUTSCALE Object Storage. Une boîte de dialogue apparaît.

  3. Spécifiez vos identifiants :

    1. Dans le champ Access Key ID, tapez l’ID de votre access key.

    2. Dans le champ Secret Access Key, tapez la secret key associée à l’ID de votre access key.

      Pour obtenir l’ID de votre access key et votre secret key pour autoriser la connexion aux API, voir Obtenir des informations sur vos access keys.

  4. Cliquez sur Login pour valider.
    Vous êtes connecté à vos ressources OOS, et vos buckets et fichiers apparaissent dans l’interface.

    • Pour configurer manuellement une connexion OOS, vous pouvez cliquer sur Open Connection . Dans ce cas, ne spécifiez pas "https" lorsque vous tapez le endpoint qui correspond à la Région que vous voulez utiliser.

    • Vous pouvez désactiver les notifications "Bonjour" de Cyberduck en exécutant la commande suivante dans votre invite de commandes :

      $ (echo rendezvous.enable=false) > "%AppData%\Cyberduck\default.properties"

      Un fichier default.properties est créé dans votre dossier Cyberduck et la propriété nécessaire pour désactiver les notifications "Bonjour" y est ajoutée.
